Перечень команд:


/?                        - справка

copy con file.txt                - создание файла

copy file1.txt file2.txt        - копирование файла

copy file1+file2 fileAll        - соединение файлов

copy /b file nul                - проверка считываемости

move /y file или dir        - перемещение или переименование

del file.txt                - удалить файл

ren file.txt nFile.txt        - переименовать файл

filefind file                - поиск файла

a: c: d:                        - смена диска

cd [каталог]                - открыть каталог

cd..                        - поднятся в древе каталогов на уровень вверх

dir /p                        - вывод оглавления с паузами

md [каталог]                - создать каталог

rd [каталог]                - удалить пустой каталог

delfree [каталог]                - удалить каталог

type file                        - вывод файла на экран

cls                        - очистить экран

copy /b file.txt prn        - вывод на печеть

format c:                - форматировать диск

label c:                        - создать метку на диск

date                        - дата

time                        - время

sysinfo                        - информация о пк

ipconfig                        - информация о сети

ping 192.168.0.1        - пинг хоста
