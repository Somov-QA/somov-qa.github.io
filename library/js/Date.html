Date

Date() - возвращает текущую дату и время.

getDate() - возвращает день месяца.

getDay() - возвращает число, представляющее день недели.

getMonth() - возвращает месяц, как число.

getFullYear() - возвращает четырехзначный номер, представляющий год.

getYear() -  возвращает год, как двухзначным или четырехзначным номер.

getHours() - возвращает час.

getMinutes() - возвращает минуты.

getSeconds() - возвращает секунды.

getMilliseconds() - возвращает миллисекунды

getTime() - возвращает количество миллисекунд, прошедших с полуночи 1 января 1970 года.

getTimezoneOffset() - возвращает разницу в минутах между местным временем и временем по Гринвичу (GMT).

getUTCDate() - возвращает день месяца согласно UTC.

getUTCDay() -  возвращает число, представляющее день недели UTC.

getUTCMonth() - возвращает число, которое представляет месяц UTC.

getUTCFullYear() - возвращает четырехзначный номер, представляющий год UTC.

getUTCHours() - возвращает час времени UTC.

getUTCMinutes() - вовзращает минуты UTC.

getUTCSeconds() - возвращает секунды UTC.

getUTCMilliseconds() - возвращает миллисекунды UTC.

parse(datestring) - принимает строку даты и возвращает количество миллисекунд, прошедших с полуночи 1 января 1970 года.

setDate(day) - установить день месяца.

setMonth(month,day) - установить месяц.

setFullYear(year,month,day)  - установка даты.

setYear(year) - установки года.

setHours(hour,min,sec,millisec) - установка часов заданного времени.

setMinutes(min,sec,millisec) - установка минут

setSeconds(sec,millisec) - установка секунд

setMilliseconds(millisec) - установка миллисекунд.

setTime(millisec) - расчет даты и времени путем сложения или вычитания определенного количества миллисекунд до / с полуночи 1 января 1970 года.

setUTCDate(day) - установить день месяца UTC.

setUTCMonth(month,day) - установить месяц UTC.

setUTCFullYear(year,month,day) - установить дату UTC.

setUTCHours(hour,min,sec,millisec) - установить час UTC.

setUTCMinutes(min,sec,millisec) - установить минуты UTC.

setUTCSeconds(sec,millisec) - установить секунды UTC.

setUTCMilliseconds(millisec) - установить миллисекунды UTC.

toSource() - метод представляет собой исходный код объекта.

toString() - преобразует в строку.

toGMTString() - преобразует дату GMT в строку.

toUTCString() - преобразует дату UTC в строку.

toLocaleString() - преобразует дату по местному времени в строку.

UTC(year,month,day,hours,minutes,seconds,ms) - принимает дату и возвращает количество миллисекунд, прошедших с полуночи 1 января 1970 года.

valueOf() - возвращает примитивное значение объекта Date.

constructor - свойство конструктора является ссылкой на функцию, которая создала объект.

prototype - позволяет добавлять свойства и методы к объекту.
