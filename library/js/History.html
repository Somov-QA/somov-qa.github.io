BOM History

Объект история содержит URL, посещенных пользователем (в окне браузера).

Объект история является частью объекта окна, и доступ к свойствам через window.history.


Свойства:

length - Возвращает количество URL-адресов в списке истории


Методы:

back() - загружает предыдущий URL в списке хронологии

forward() - загружает следующий URL в списке хронологии

go(number|URL) - загружает определенный URL из списка истории

<!DOCTYPE html>
<html>
<head>
   <title></title>
   <meta charset="utf-8" />
   <style type="text/css">
           html, body {
               border: 0;
               padding: 0;
               margin: 0;
           }
       </style>

   <script type="text/javascript">
           function onLoad()
           {
               console.log('length:', history.length);
               window.history.go(-1);
               window.history.back();
               window.history.forward();
           }
       </script>
</head>
<body onload="onLoad()">

</body>
</html>
