Методы работы с массивами

       pop()                - получить последний элемент

shift()                - получить первый элемент

       push()                - добавить в конец

       unshift()                - добавить в начало

       reverse()        - обратный массив

       slice()                - выбор из диапазона

       splice()                - удаление из диапазона
