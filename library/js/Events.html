DOM Events

HTML DOM события позволяют JavaScript регистрировать различные обработчики событий элементов в HTML-документе.


Mouse Events:

onclick - Событие происходит, когда пользователь нажимает на элемент

oncontextmenu - Событие происходит, когда щелчке правой кнопкой мыши на элементе, чтобы открыть контекстное меню

ondblclick - Событие происходит, когда пользователь дважды щелкает на элементе

onmousedown - Событие происходит, когда пользователь нажимает кнопку мыши на элементе

onmouseenter - Событие происходит, когда указатель перемещается на элемент

onmouseleave - Событие происходит, когда указатель перемещается из элемента

onmousemove - Событие происходит, когда указатель перемещается в то время как он находится над элементом

onmouseover - Событие происходит, когда указатель перемещается на элемент, или на одном из своих детей

onmouseout - Событие происходит, когда пользователь перемещает указатель мыши из элемента, или из одного из своих детей

onmouseup - Событие происходит, когда пользователь отпускает кнопку мыши над элементом


Keyboard Events:

onkeydown - Событие происходит, когда пользователь нажимает клавишу

onkeypress - Событие происходит, когда пользователь нажимает клавишу

onkeyup - Событие происходит, когда пользователь отпускает клавишу


Frame/Object Events:

onabort - Событие происходит, когда загрузка ресурса была прервана

onbeforeunload - Событие происходит до того, как документ собирается быть выгружен

onerror - Событие происходит, когда возникает ошибка при загрузке внешнего файла

onhashchange - Событие происходит, когда произошло изменения в анкерной части URL

onload - Событие происходит, когда объект загружен

onpageshow - Событие происходит, когда пользователь переходит на веб-страницу

onpagehide - Событие происходит, когда пользователь покидает веб-страницы

onresize - Событие происходит, когда вид документа изменен

onscroll - Событие происходит, когда полоса прокрутки Элементом в настоящее время прокручивается

onunload - Событие происходит один раз страница разгрузил (для <body>)


Form Events:

onblur - Событие происходит, когда элемент теряет фокус

onchange - событие возникает, когда содержание элемента формы, выбор, или проверяемого состояния изменились (для <input>, <keygen>, <select>, и <textarea>)

onfocus - Событие происходит, когда элемент получает фокус

onfocusin - Событие происходит, когда элемент собирается получить фокус

onfocusout - Событие происходит, когда элемент собирается потерять фокус

oninput - Событие происходит, когда элемент получает ввод данных пользователем

oninvalid - Событие происходит, когда элемент является недействительным

onreset - Событие происходит, когда форма сбрасывается

onsearch - Событие происходит, когда пользователь пишет что-то в поле поиска (для <input="search">)

onselect - Событие происходит после того, как пользователь выбирает некоторый текст (для <input> и <textarea>)

onsubmit - Событие происходит, когда форма была отправлена


Drag Events:

ondrag - Событие происходит, когда элемент перетаскивается

ondragend - Событие происходит, когда пользователь завершает перетаскивание элемент

ondragenter - Событие происходит, когда тащили элемент попадает в цель снижения

ondragleave - Событие происходит, когда тащили элемент покидает цель снижения

ondragover - Событие происходит, когда тащили элемент над целевым объектом перетаскивания

ondragstart - Событие происходит, когда пользователь начинает перетаскивать элемент

ondrop - Событие происходит, когда тащили элемент падает на цель перетаскивания


Clipboard Events:

oncopy - Событие происходит, когда пользователь копирует содержимое элемента

oncut - Событие происходит, когда пользователь обрезает содержимое элемента

onpaste - Событие происходит, когда пользователь вставляет некоторое содержание в элементе


Print Events:

onafterprint - Событие происходит, когда страница начала печати, или если окно печати диалог был закрыт

onbeforeprint - Событие происходит, когда страница о печатаемых


Media Events:

onabort - Событие происходит, когда нагрузка в СМИ прерывается

oncanplay - Событие происходит, когда браузер может начать воспроизведение носителя (когда он буферном достаточно, чтобы начать)

oncanplaythrough - Событие происходит, когда браузер может воспроизводить через средства массовой информации без остановки для буферизации

ondurationchange - Событие происходит, когда длительность СМИ изменяется

onemptied - Событие происходит, когда что-то плохое происходит, и медиа-файл вдруг недоступен (например, неожиданно разъединяет)

onended - Событие происходит, когда носитель имеет достигают конца (полезно для сообщений типа "спасибо за прослушивание")

onerror - Событие происходит, когда произошла ошибка во время загрузки медиа-файла

onloadeddata - Событие происходит, когда носители данных загружается

onloadedmetadata - Событие происходит, когда мета-данные (например, размеров и продолжительности) загружаются

onloadstart - Событие происходит, когда браузер начинает искать для указанного средства массовой информации

onpause - Событие происходит, когда СМИ приостановлена ​​либо пользователем, либо программно

onplay - Событие происходит, когда было запущено СМИ или больше не приостановлена

onplaying - Событие происходит, когда среда играет после того, как был остановлен или остановлен для буферизации 3

onprogress - Событие происходит, когда браузер находится в процессе получения мультимедийных данных (загрузка носителя)

onratechange - Событие происходит, когда скорость воспроизведения СМИ изменяется

onseeked - Событие происходит, когда пользователь заканчивает перемещение / пропуск в новую позицию в средствах массовой информации

onseeking - Событие происходит, когда пользователь начинает движение / пропуск в новую позицию в средствах массовой информации

onstalled - Событие происходит, когда браузер пытается получить мультимедийные данные, но данные не доступны

onsuspend - Событие происходит, когда браузер намеренно не получать мультимедийные данные

ontimeupdate - Событие происходит, когда позиция воспроизведения изменилась (например, когда пользователь быстро вперед к другой точке в средствах массовой информации)

onvolumechange - Событие происходит, когда объем средств массовой информации изменилось (включает в себя установку громкости на "немой")

onwaiting - Событие происходит, когда СМИ приостановлена, но ожидается возобновление (например, когда носитель делает паузу в буфер больше данных)


Animation Events:

animationend - Событие происходит, когда CSS анимация завершена

animationiteration - Событие происходит, когда CSS анимация повторяется

animationstart - Событие происходит, когда начал CSS-анимации


Transition Events:

transitionend - Событие происходит, когда переход завершен CSS


Server-Sent Events:

onerror - Событие происходит при возникновении ошибки с источником событий

onmessage - Событие происходит, когда сообщение получено через источник событий

onopen - Событие происходит, когда соединение с источником событий открыт


Misc Events:

onmessage - Событие происходит, когда сообщение получено через или от объекта (WebSocket, Web Worker, Event Source or a child frame or a parent window)

onmousewheel - устаревший. Используйте событие onwheel вместо

ononline - Событие происходит, когда браузер начинает работать в режиме онлайн

onoffline - Событие происходит, когда браузер начинает работать в автономном режиме

onpopstate - Событие происходит при изменении истории окна

onshow - Событие происходит, когда элемент <menu> отображается в виде контекстного меню

onstorage - Событие происходит, когда область Web Storage обновляется

ontoggle - Событие происходит, когда пользователь открывает или закрывает элемент <details>

onwheel - Событие происходит, когда колесо мыши катится вверх или вниз над элементом


Touch Events:

ontouchcancel - Событие происходит, когда прикосновение прерывается

ontouchend - Событие происходит, когда палец удаляется с сенсорным экраном

ontouchmove - Событие происходит, когда палец тащили по экрану

ontouchstart - Событие происходит, когда палец помещают на сенсорном экране


Event Object:


Константы:

CAPTURING_PHASE - Текущая фаза событие является фаза захвата (1)

AT_TARGET - Текущее событие в целевой фазе, т.е. она оценивается на цели события (2)

BUBBLING_PHASE - Текущая фаза событие является фаза восходящей цепочки (3)


Свойства:

bubbles - Возвращает ли не конкретное событие является событием восходящей цепочки

cancelable - Возвращает или не может быть его действие по умолчанию предотвратили событие

currentTarget - Возвращает элемент, событие слушателей вызвал событие

defaultPrevented - Возвращает был ли или не вызывается метод preventDefault() для события

eventPhase - Возвращает какой фазе потока событий, в настоящее время оценивается

isTrusted - Возвращает ли или не является доверенным событие

target - Возвращает элемент, который вызвал событие

timeStamp - Возвращает время (в миллисекундах относительно эпохи), на котором был создан событие

type - Возвращает имя события

view - Возвращает ссылку на объект Window, где случилась событие


Методы:

preventDefault() - Отменяет событие, если оно отменяемый, а это означает, что действие по умолчанию, которое принадлежит к событию не произойдет

stopImmediatePropagation() - Предотвращает других слушателей одного и того же события от призыва

stopPropagation() - предотвращает дальнейшее распространение события во время потока событий


MouseEvent Object:

altKey - Возвращает клавишу Alt, была ли нажата клавиша "ALT", когда событие мыши было вызвано

button - Возвращает, какая кнопка мыши была нажата, когда событие мыши было вызвано

buttons - Возвращает какие кнопки мыши были нажаты, когда событие мыши было вызвано

clientX - Возвращает горизонтальную координату указателя мыши, относительно текущего окна, когда событие мыши было вызвано

clientY - Возвращает вертикальную координату указателя мыши, относительно текущего окна, когда событие мыши было вызвано

ctrlKey - Возвращает ли нажата клавиша "CTRL", когда событие мыши было вызвано

detail - Возвращает число, указывающее, сколько раз была нажата мышь

metaKey - Возвращает ли нажата клавиша "META", когда событие было вызвано

pageX - Возвращает горизонтальной координаты курсора мыши, по отношению к документу, когда событие мыши было вызвано

pageY - Возвращает вертикальную координату указателя мыши, по отношению к документу, когда событие мыши было вызвано

relatedTarget - Возвращает элемент, связанный с элементом, вызвавшего событие мыши

screenX - Возвращает горизонтальной координаты курсора мыши, по отношению к экрану, когда событие было вызвано

screenY - Возвращает вертикальную координату указателя мыши, по отношению к экрану, когда событие было вызвано

shiftKey - Возвращает ли нажата клавиша "SHIFT", когда событие было вызвано

which - Возвращающий какая кнопка мыши была нажата, когда событие мыши было вызвано


KeyboardEvent Object:

altKey - Возвращает клавишу Alt, была ли нажата клавиша "ALT", когда ключ событие было вызвано

ctrlKey - Возвращает ли нажата клавиша "CTRL", когда ключ событие было вызвано

charCode - Возвращает код символа Unicode ключа, инициировавшего событие OnKeyPress

key - возвращает ключ значение ключа, представленного события

keyCode - Возвращает скан-кода код символа Unicode ключа, инициировавшего событие OnKeyPress или код ключа Unicode ключа, инициировавшего OnKeyDown или OnKeyUp событие

location - Возвращает расположение клавиши на клавиатуре или устройства

metaKey - Возвращает ли нажата клавиша "мета", когда ключ событие было вызвано

shiftKey - Возвращает ли нажата клавиша "SHIFT", когда ключ событие было вызвано

which - Возвращает код символа Unicode ключа, инициировавшего событие OnKeyPress или код ключа Unicode ключа, инициировавшего OnKeyDown или OnKeyUp событие


HashChangeEvent Object:

newURL - Возвращает URL документа, после того, как хэш был изменен

oldURL - Возвращает URL документа, до того, как хэш был изменен


PageTransitionEvent Object:

persisted - Возвращает был ли веб-страницы кэшируются браузером


FocusEvent Object:

relatedTarget - Возвращает элемент, связанный с элементом, вызвавшего событие


AnimationEvent Object:

animationName - Возвращает имя анимации

elapsedTime - Возвращает количество секунд анимации было работает


TransitionEvent Object:

propertyName - Возвращает имя свойства CSS, связанное с переходом

elapsedTime - Возвращает количество секунд переход был запущен


WheelEvent Object:

deltaX - Возвращает горизонтальную величину прокрутки колеса мыши (ось х)

deltaY - Возвращает вертикальное количество прокрутки колеса мыши (ось у)

deltaZ - Возвращает количество прокрутки колеса мыши для оси

deltaMode - Возвращает число, представляющее собой единицу измерения для значений дельта (пикселей, линий или страниц)

<html>
<head>
   <title></title>

       html, body {
           border: 0;
           padding: 0;
           margin: 0;
       }
   </style>

   <script type="text/javascript">
       function onLoad()
       {
          var event = new Event("click");
          myButton.dispatchEvent(event);
       }

       function onClick()
       {
           alert('Нажата кнопка!');
       }
   </script>
</head>
<body onload="onLoad()">
   <button id="myButton" onclick="onClick()">Автоклик</button>
</body>
</html>
