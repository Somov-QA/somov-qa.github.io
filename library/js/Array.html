Array

concat(arrayX,arrayX,......,arrayX) - соединения двух или более массивов (возвращает новый массив).

join(separator) - поместить все элементы в строку разделив их значением separator (например ".").

pop() - удаление и возврат последнего элемента массива.

push(newelement1,newelement2,....,newelementX) - добавляем элемент в конец массива и возвращает длину.

reverse() - изменения порядка элеменов в массиве.

shift() - удаляет и возвращает первый элемент массива.

slice(start,end) - возвращает выбранный элемент из массива.

sort(sortby) - сортировка элементов массива.

splice(index,howmany,element1,.....,elementX) - удаление и добавление новых элементов в массив.

toSource() - метод представляет собой исходный код объекта.

toString() - преобразует массив в строку.

unshift(newelement1,newelement2,....,newelementX) - добавляет один или несколько элементов в начало массива и возвращает длину.

valueOf() - возвращает примитивное значение объекта Array.

constructor - свойство конструктора является ссылкой на функцию, которая создала объект.

input

length - наборы свойств или возвращает количество элементов в массиве.

prototype - позволяет добавлять свойства и методы к объекту.
