Attr Object.

В HTML DOM объект Attr представляет атрибут HTML.

Атрибут HTML всегда принадлежит к HTML-элементу.


NamedNodeMap Object.

В HTML DOM объект NamedNodeMap представляет собой неупорядоченный инкассации элементов узлов атрибутов.

Узлы в NamedNodeMap можно обращаться по имени или по индексу (номер).


Свойства и методы:

attr.isId - Возвращает истину, если атрибут имеет тип Id, в противном случае она возвращает ложь

attr.name - Возвращает имя атрибута

attr.value - Устанавливает или возвращает значение атрибута

attr.specified - Возвращает истину, если был указан атрибут, в противном случае она возвращает ложь

 

nodemap.getNamedItem() - Возвращает указанный узел атрибута из NamedNodeMap

nodemap.item() - Возвращает узел атрибута по указанному индексу в NamedNodeMap

nodemap.length - Возвращает количество узлов атрибутов в NamedNodeMap

nodemap.removeNamedItem() - Удаляет указанный узел атрибута

nodemap.setNamedItem() - Устанавливает заданный узел атрибута (по имени)


DOM 4 Внимание!

В W3C DOM ядро, Attr (атрибут) объект наследует все свойства и методы из объекта Node.

В DOM 4, объект Attr больше не наследует от Node.

Для будущего качества кода, вы должны избегать использования свойств и методов объекта узла на объекты атрибутов:


attr.appendChild() - Атрибуты не имеют дочерних узлов

attr.attributes - Атрибуты не имеют атрибутов

attr.baseURI - использовать вместо document.baseURI

attr.childNodes - Атрибуты не имеют дочерних узлов

attr.cloneNode() - Получить или установить вместо attr.value

attr.firstChild - Атрибуты не имеют дочерних узлов

attr.hasAttributes() - Атрибуты не имеют атрибутов

attr.hasChildNodes - Атрибуты не имеют дочерних узлов

attr.insertBefore() - Атрибуты не имеют дочерних узлов

attr.isEqualNode() - не делает никакого смысла

attr.isSameNode() - не делает никакого смысла

attr.isSupported() - всегда верно

attr.lastChild - Атрибуты не имеют дочерних узлов

attr.nextSibling - Атрибуты не имеют братьев и сестер

attr.nodeName - Используйте attr.name вместо

attr.nodeType - Это всегда 2 (ATTRIBUTE_NODE)

attr.nodeValue - Используйте attr.value вместо

attr.normalize() - Атрибуты не могут быть нормализованы

attr.ownerDocument - Это всегда ваш HTML-документ

attr.ownerElement - Это HTML-элемент, который вы использовали для доступа к атрибуту

attr.parentNode - Это HTML-элемент, который вы использовали для доступа к атрибуту

attr.previousSibling - Атрибуты не имеют братьев и сестер

attr.removeChild - Атрибуты не имеют дочерних узлов

attr.replaceChild - Атрибуты не имеют дочерних узлов

attr.textContent - Используйте attr.value вместо

               padding: 0;
               margin: 0;
           }
       </style>

   <script type="text/javascript">
           function onLoad()
           {
               var div = document.getElementById("divID");
               for(var i = 0; i < div.attributes.length; i++){
                   console.log("Div:", div.attributes[i].name, "value:", div.attributes[i].value);
               }

               console.log("getNamedItem", div.attributes.getNamedItem('name').value);
               console.log("item[0]", div.attributes.item(0).nodeName);

               div.attributes.removeNamedItem("name");

               var newItem = document.createAttribute("class");
               newItem.value = "democlass"
               div.attributes.setNamedItem(newItem);
           }
       </script>
</head>
<body onload="onLoad()">
<div name="divName" id="divID"></div>

</body>
