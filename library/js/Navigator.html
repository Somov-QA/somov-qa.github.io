BOM Navigator

Объект навигатора содержит информацию о браузере.


Коллекции:

plugins[] - возвращает ссылку на все встроенные объекты в документе


Свойства:

appCodeName - Возвращает кодовое имя браузера

appMinorVersion - Возвращает минорную версию браузера

appName - Возвращает имя браузера

appVersion - Возвращает информацию о версии браузера

browserLanguage - Возвращает Язык браузера текущий язык браузера

cookieEnabled - Определяет, включены ли куки в браузере

geolocation - Возвращает объект Geolocation, который может использоваться, чтобы определить местонахождение положение пользователя

language - Возвращает язык браузера

cpuClass - Возвращает класс CPU системы браузера

onLine - Определяет браузер онлайн

platform - Возвращает, для которой платформа скомпилированные браузер

product - Возвращает имя двигателя браузера

systemLanguage - Возвращает язык по умолчанию, используемый ОС

userAgent - Возвращает заголовок агента пользователя, отправленный браузером на сервер

userLanguage - Возвращает параметр естественного языка ОС '


Методы:

javaEnabled() - Определяет, будет ли или нет быть включен Java-браузер

taintEnabled() - Удалены в JavaScript версии 1.2. Указывает, был ли включен браузер данные загрязненность
