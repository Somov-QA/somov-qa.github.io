BOM Location

Объект местоположения содержит информацию о текущем URL.

Объект расположение является частью объекта окна, и доступ через свойство window.location.


Свойства:

hash - Устанавливает или возвращает анкерную часть (#) от URL

host - Устанавливает или возвращает имя хоста и номер порта в URL

hostname - Задает или возвращает имя хоста в URL

href - Устанавливает или возвращает весь URL

origin - Возвращает протокол, имя хоста и номер порта в URL

pathname - Устанавливает путь к файлу или возвращает имя пути в URL

port - Устанавливает или возвращает номер порта URL

protocol - Устанавливает или возвращает протокол в URL

search - Устанавливает или возвращает строку запроса часть URL


Методы:

assign(URL) - Загружает новый документ

reload(forceGet) - Перезагружает текущий документ

replace(newURL) - Заменяет текущий документ на новым

<!DOCTYPE html>
<html>
<head>
   <title></title>
   <meta charset="utf-8" />
   <style type="text/css">
           html, body {
               border: 0;
               padding: 0;
               margin: 0;
           }
       </style>

   <script type="text/javascript">
           function onLoad()
           {
               console.log('hash:', location.hash);
               console.log('host:', location.host);
               console.log('hostname:', location.hostname);
               console.log('href:', location.href);
               console.log('origin:', location.origin);
               console.log('pathname:', location.pathname);
               console.log('port:', location.port);
               console.log('protocol:', location.protocol);
               console.log('search:', location.search);
               location.assign("http://www.w3schools.com");
               location.reload();
               location.replace("http://www.w3schools.com");
           }
       </script>
</head>
<body onload="onLoad()">

</body>
</html>
