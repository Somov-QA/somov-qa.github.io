DOM Base

Элемент <base> определен внутри контейнера <head> и инструктирует браузер относительно полного базового адреса текущего документа. Тег <base> предназначен для документов, в которых используется относительный адрес и эти документы могут переноситься в другую папку или даже на другой компьютер без потери связи.


Свойства:

href - Устанавливает или возвращает базовый URL для документа

id - Устанавливает или возвращает идентификатор базового объекта

target - Устанавливает или возвращает целевой по умолчанию фрейм или окно

<!DOCTYPE html>
<html>
<head>
   <title></title>
   <meta charset="utf-8" />
   <base id="myBase" href="http://somov.inf.ua" target="_blank" />
   <style type="text/css">
           html, body {
               border: 0;
               padding: 0;
               margin: 0;
           }
   </style>

   <script type="text/javascript">
           function onLoad()
           {
               var myBase = document.getElementById('myBase');
               console.log("myBase:", myBase.id, myBase.href, myBase.target);
           }
   </script>
</head>
<body onload="onLoad()">

</body>
</html>
