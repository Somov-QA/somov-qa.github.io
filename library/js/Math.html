Math

Math.abs(x) - абсолютное значение числа.

Math.acos(x) - арккосинус числа как числовое значение значение от 0 до PI радиан.

Math.asin(x) - арксинус числа как числовое значение между -PI / 2 и PI / 2 радиан.

Math.atan(x) - арктангенс числа как числовое значение в диапазоне от -pi / 2 и PI / 2 радиан.

Math.atan2(x,y) - угол тета ап (х, у) точки как числовое значение в диапазоне от -pi / 2 и PI / 2 радиан.

Math.ceil(x) - значение числа округляется в большую сторону до ближайшего целого числа.

Math.cos(x) - косинус числа.

Math.exp(x) - значение E в степени x, где Е постоянная Эйлера (приблизительно 2,7183) и х число передается к нему.

Math.floor(x) - значение числа округлением в меньшую до ближайшего целого числа.

Math.log(x) - натуральный логарифм (base E) числа.

Math.max(x,y) - число с наибольшим значением.

Math.min(x,y) - число с наименишим значением.

Math.pow(x,y) - значение х в степени у.

Math.random() - случайное число от 0 до 1.

Math.round(x) - округляет число до ближайшего целого числа.

Math.sin(x) - синус числа.

Math.sqrt(x) - квадратный корень из числа.

Math.tan(x) - тангенс угла.

toSource() - метод представляет собой исходный код объекта.

valueOf() - возвращает примитивное значение объекта Math.

constructor - свойство конструктора является ссылкой на функцию, которая создала объект.

prototype - позволяет добавлять свойства и методы к объекту.

E - возвращает константу Эйлера (прибл. 2,718)

LN2 - возвращает натуральный логарифм 2 (прибл. 0,693)

LN10 - возвращает натуральный логарифм 10 (прибл. 2,302)

LOG2E - возвращает базовый-2 логарифм E (прибл. 1,442)

LOG10E - возвращает базовый-10 логарифм E (прибл. 0,434)

PI - возвращает значение Пи (ок. 3,14159)

SQRT1_2 - возвращает квадратный корень из 1/2 (прибл. 0,707)

SQRT2 - возвращает квадратный корень из 2 (прибл. 1,414)
