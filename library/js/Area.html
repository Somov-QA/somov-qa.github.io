DOM Area

Каждый элемент <area> определяет активные области изображения, которые являются ссылками.


Свойства:

accessKey - Устанавливает или возвращает клавишу клавиатуры для доступа к объекту области

alt - Устанавливает или возвращает альтернативный текст для браузеров, которые не могут показать область

coords - Устанавливает или возвращает разделенный запятыми список длин, который определяет активная область

hash - Устанавливает или возвращает анкерную часть URL (начинается с #)

host - Устанавливает или возвращает имя хоста и порт URL

hostname - Устанавливает или возвращает имя сервера и доменное имя / IP-адрес URL

href - Устанавливает или возвращает весь URL

id - Устанавливает или возвращает идентификатор объекта области (В IE 4 это свойство только для чтения)

noHref - Устанавливает или возвращает, если область должна быть неактивным

pathname - Задает или возвращает имя пути URL

port - Задает или возвращает URL-адрес порта, на котором используется сервер

protocol - Устанавливает или возвращает протокол URL (в том числе первого двоеточия) e.g "HTTP:", "https:"

search - Устанавливает или возвращает раздел URL строки запроса (включая знак вопроса (?))

shape - Устанавливает или возвращает форму области

tabIndex - Устанавливает или возвращает порядок табуляции объекта области

target - Устанавливает или возвращает имя целевого URL ссылки в


События:

onClick - Выполняет некоторый код, когда происходит событие Нажмите

onDblClick - Выполняет некоторый код, когда происходит событие Двойное нажатие

onMouseOut - Выполняет некоторый код, когда происходит событие Мышь покидает пределы объекта

onMouseOver - Выполняет некоторый код, когда происходит событие Мышь входит в пределы объекта

<!DOCTYPE html>
<html>
<head>
   <meta charset="utf-8">
   <title></title>

       }
   </style>

   <script type="text/javascript">
           function onClick()
           {
               document.getElementById('image').src="image0.png"
           }

           function onDblClick(id)
           {
               document.getElementById('image').src="image" + id + ".png"
           }

           function onMouseOut()
           {
               document.getElementById('image').src="image0.png"
           }

           function onMouseOver(id)
           {
               document.getElementById('image').src="image" + id + ".png"
           }
       </script>

</head>
<body>
<div>
