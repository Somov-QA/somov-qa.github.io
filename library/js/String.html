String

anchor(anchorname) - создания HTML-якоря.

big() - большой шрифт

blink() - Отображает мигающего строку

bold() - Отображает строку жирным

charAt(index) - Возвращает символ в указанной позиции

charCodeAt(index) - Возвращает Unicode символа в указанной позиции

concat(stringX,stringX,...,stringX) - соединяет две или более строк

fixed() - Отображает строку как телетайп текст

fontcolor(color) - Отображает строку в заданном цвете

fontsize(size) - Отображает строку в указанном размере

fromCharCode(numX,numX,...,numX) - принимает заданные значения Unicode и возвращает строку

indexOf(searchvalue,fromindex) - Возвращает позицию первого вхождения заданного значения строки в строке

italics() - Отображает строку в курсивное

lastIndexOf(searchvalue,fromindex) - Возвращает позицию последнего вхождения заданного значения строки, поиск в обратном направлении от указанной позиции в строке

link() - Отображает строку как гиперссылку

match(searchvalue) - Выполняет поиск заданного значения строки в строке

replace(findstring,newstring) - заменяет некоторые символы с некоторыми другими символами в строке

search(searchstring) - Ищет строку для заданного значения

slice(start,end) - Извлекает часть строки и возвращает Извлеченный часть в новой строке

small() - Отображает строку мелким шрифтом

split(separator, howmany) - Разбивает строку на массив строк

strike() - Отображает строку с зачеркивания

sub() - Отображает строку как нижний индекс

substr(start,length) - Извлекает заданное количество символов в строке, из индекса пуска

substring(start,stop) - извлекает символы в строке между двумя указанными индексами

sup() - Отображает строку в качестве верхнего индекса

toLowerCase() - Отображает строку строчными буквами

toUpperCase() - Отображает строку заглавными буквами

toSource() - Представляет исходный код объекта

valueOf() - Возвращает примитивное значение объекта Струнный

constructor - свойство конструктора является ссылкой на функцию, которая создала объект.

length - наборы свойств или возвращает количество элементов в массиве.
