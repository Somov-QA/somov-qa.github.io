DOM Element

В HTML DOM, объект Element представляет собой HTML-элемент.

Объекты элемент может иметь дочерние узлы узлов типа элементов, текстовых узлов, или комментарий узлов.

Объект NodeList представляет собой список узлов, как коллекции HTML-элемента дочерних узлов.

Элементы могут также иметь атрибуты. Атрибуты узлов атрибутов (смотрите следующий раздел).


Свойства и методы:

accessKey - Устанавливает или возвращает атрибут Accesskey элемента

addEventListener(event, function, useCapture) - Прикрепляет обработчик события для указанного элемента

appendChild(node) - Добавляет новый дочерний узел, к элементу, в качестве последнего дочернего узла

attributes - Возвращает NamedNodeMap из атрибутов элемента

blur() - Удаляет фокус из элемента

childElementCount - Возвращает число дочерних элементов элемент имеет

childNodes - Возвращает коллекцию дочерних узлов Элемент (включая текст и узлы комментариев)

children - Возвращает коллекцию элемента дочернего элемента не давал (за исключением текста и узлы комментариев)

classList - Возвращает имя класса (ов) элемента

className - Устанавливает или возвращает значение атрибута класса элемента

click() - Имитирует щелчок мыши на элементе

clientHeight - Возвращает высоту элемента, включая отступы

clientLeft - Возвращает ширину левой границы элемента

clientTop - Возвращает ширину верхней границы элемента

clientWidth - Возвращает ширину элемента, включая отступы

cloneNode(deep) - Клоны элемент

compareDocumentPosition(node) - Сравнивает положение документа двух элементов

contains(node) - Возвращает истину, если узел является потомком узла, в противном случае ложной

contentEditable - Устанавливает или возвращает ли можно редактировать содержимое элемента или нет

dir - Устанавливает или возвращает значение атрибута dir элемента

firstChild - Возвращает первый дочерний узел элемента

firstElementChild - Возвращает первый дочерний элемент элемента

focus() - Передает фокус элементу

getAttribute(attributename) - Возвращает заданное значение атрибута узла элемента

getAttributeNode(attributename) - Возвращает указанный узел атрибута

getElementsByClassName(classname) - Возвращает коллекцию всех дочерних элементов с указанным именем класса

getElementsByTagName(tagname) - Возвращает коллекцию всех дочерних элементов с указанным именем тега

getFeature() - Возвращает объект, который реализует API-интерфейсы для указанной функции

hasAttribute(attributename) - Возвращает истину, если элемент имеет указанный атрибут, в противном случае ложной

hasAttributes() - Возвращает истину, если элемент имеет какие-либо атрибуты, иначе ложь

hasChildNodes() - Возвращает истину, если элемент имеет дочерние узлы, в противном случае ложной

id - Устанавливает или возвращает значение атрибута идентификатор элемента

innerText - Устанавливает или возвращает содержимое элемента текст

innerHTML - Устанавливает или возвращает содержимое элемента

insertBefore(newnode, existingnode) - Вставляет новый дочерний узел перед указанным, существующий, дочерний узел

isContentEditable - Возвращает истину, если содержание элемента можно редактировать, в противном случае ложной

isDefaultNamespace(namespaceURI) - Возвращает истину, если указанный NamespaceURI по умолчанию, в противном случае ложной

isEqualNode(node) - Проверяет, если два элемента равны

isSameNode(node) - Проверяет, если два элемента и тот же узел

isSupported(feature, version) - Возвращает истину, если указанная функция поддерживается на элементе

lang - Устанавливает или возвращает значение атрибута Ланг элемента

lastChild - Возвращает последний дочерний узел элемента

lastElementChild - Возвращает последний дочерний элемент элемента

namespaceURI - Возвращает URI пространства имен элемента

nextSibling - Возвращает следующий узел на том же уровне дерева узла

nextElementSibling - Возвращает следующий элемент на том же уровне дерева узлов

nodeName - Возвращает имя узла

nodeType - Возвращает тип узла узла

nodeValue - Устанавливает или возвращает значение узла

normalize() - соединяет смежных текстовых узлов и удаляет пустые текстовые узлы в элементе

offsetHeight - Возвращает высоту элемента, включая отступы, границы и скроллинга

offsetWidth - Возвращает ширину элемента, включая отступы, границы и скроллинга

offsetLeft - Возвращает горизонтальное смещение позиции элемента

offsetParent - Возвращает смещение контейнера элемента

offsetTop - Возвращает вертикальное смещение позиции элемента

ownerDocument - Возвращает корневой элемент (документ объекта) для элемента

parentNode - Возвращает родительский узел элемента

parentElement - Возвращает родительский элемент узла элемента

previousSibling - Возвращает предыдущий узел на том же уровне дерева узлов

previousElementSibling - Возвращает предыдущий элемент на том же уровне дерева узла

querySelector(CSS selectors) - Возвращает первый дочерний элемент, соответствующий селектор CSS указан (ы) элемента

querySelectorAll(CSS selectors) - Возвращает все дочерние элементы, который соответствует селектор CSS указано (ы) элемента

removeAttribute(attributename) - Удаляет указанный атрибут из элемента

removeAttributeNode(attributenode) - Удаляет указанный узел атрибута, и возвращает удаленный узел

removeChild(node) - Удаляет дочерний узел из элемента

replaceChild(newnode, oldnode) - Заменяет дочерний узел в элементе

removeEventListener(event, function, useCapture) - Удаляет обработчик событий, который был присоединен с addEventListener()

scrollHeight - Возвращает всю высоту элемента, включая отступы

scrollIntoView(alignTo) - прокручивают указанный элемент в видимой области окна браузера

scrollLeft - Устанавливает или возвращает количество пикселей содержимого элемента прокручивается по горизонтали

scrollTop - Устанавливает или возвращает количество пикселей содержимого элемента прокручивается по вертикали

scrollWidth - Возвращает всю ширину элемента, включая отступы

setAttribute(attributename, attributevalue) - Устанавливает или изменяет указанный атрибут, к указанному значению

setAttributeNode(attributenode) - Устанавливает или изменяет указанный узел атрибута

style - Устанавливает или возвращает значение атрибута стиля элемента

tabIndex - Устанавливает или возвращает значение атрибута TabIndex элемента

tagName - Возвращает имя тега элемента

textContent - Устанавливает или возвращает текстовое содержимое узла и его потомков

title - Устанавливает или возвращает значение целевого атрибута элемента

toString() - Преобразует элемент в строку

 

nodelist.item(index) - Возвращает узел по указанному индексу в NodeList

nodelist.length - Возвращает количество узлов в NodeList

<!DOCTYPE html>
<html>
<head>
   <title></title>
   <meta charset="utf-8" />
   <style type="text/css">
           html, body {
               border: 0;
               padding: 0;
               margin: 0;
           }
       </style>

   <script type="text/javascript">
           function onLoad()
           {
               var btn = document.createElement("button");
               var titleBtn = document.createTextNode("Click me");
               btn.appendChild(titleBtn);
               btn.addEventListener("click", onClickButton);
               document.body.appendChild(btn);

               var h1 = document.createElement("h1");
               var att = document.createAttribute("id");
               att.value = "textH1";
               h1.setAttributeNode(att);
               document.body.appendChild(h1);

           }

           function onClickButton()
           {
               var h1 = document.getElementById("textH1");
               if(h1.innerText == ""){
                   h1.innerText = "Hello World!!!"
               } else {
                   document.querySelector("#textH1").innerHTML = "Heppy New Year!";
                   document.getElementsByTagName("button")[0].removeEventListener("click", onClickButton);
               }
               console.log("button click");
           }
       </script>
</head>
<body onload="onLoad()">

</body>
</html>
