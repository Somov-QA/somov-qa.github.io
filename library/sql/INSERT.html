Добавление данных INSERT

Однострочная инструкция INSERT - позволяет добавить в таблицу одну новую строку


INSERT INTO table1 (elem1, elem2, elem3)

VALUE ('text', 150.50, '2017-01-01');


Могострочная инструкция INSERT - обеспечивает извлечение строк из одной части базы данных и добавление из в другую таблицу.


INSERT INTO table1 (elem1, elem2, elem3)

SELECT elem1, elem2, elem3

FROM table2 WHERE elem4 < '2008-01-01';


Пакетная загрузка - утилита для добавления в таблицу данных из внешнего файла.
