Условия отбора в подзапросе


Сравнение с результатом подзапроса.

Значение выражения сравнивается с одним значением, которое возвращается подзапросом.


SELECT element1 FROM table1

WHERE element2 >= (SELECT elem1 FROM table2 WHERE elem2 = 'Moscow');



Проверка на принадлежность результатам подзапроса. (IN)

Значение выражения проверяется на равенство одному из множества значений, которые возвращаются подзапросом.


SELECT element1 FROM table1

WHERE element2 IN (SELECT elem1 FROM table2 WHERE elem2 > 1000);


SELECT element1 FROM table1

WHERE element2 NOT IN (SELECT elem1 FROM table2 WHERE elem2 > 1000);



Проверка на существование. (EXISTS)

Проверяется наличие строк в таблице результатов подзапроса.


SELECT element1 FROM table1

WHERE EXISTS (SELECT elem1 FROM table2 WHERE elem2 > 25000.00);



Многократное сравнение. (ANY и ALL)

Значение выражения сравнивается с каждым из множеств значений, которые возвращаются nодзапросом.


ANY - проверяет есть ли хоть одно совпадение из столбцом данных

ALL - проверяет чтобы все значения совпадали с столбцом данных


SELECT element1 FROM table1

WHERE element2 < ANY (SELECT elem1 FROM table2 WHERE elem2 = elem3);


SELECT element1 FROM table1

WHERE element2 < ALL (SELECT elem1 FROM table2 WHERE elem2 = elem3);
