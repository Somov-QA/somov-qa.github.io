ANY и ALL – Многократное сравнение

Значение выражения сравнивается с каждым из множеств значений, которые возвращаются подзапросом.

ANY – есть ли хоть одно совпадение из столбцов данных

ALL – все значения должны совпадать с столбцом данных.


SELECT table1.elem1 FROM table1 WHERE table1.elem1 < ANY (

SELECT  table2.elem2 FROM table2 WHERE table2.elem1 < 10

);
