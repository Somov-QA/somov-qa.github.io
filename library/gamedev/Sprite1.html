Текстуры

Класс хранит перечень загружаемых текстур в игре

class Images{

     public static Background:string = "background";

     public static preloadList: string[] = [

           Images.Background

     ];

}


Загрузка текстур

this.game.state.start(Fabrique.PreSplash.Name, true, false, {

     preloadHandler: (): void => {

           Images.preloadList.forEach((assetName:string) =>{

                 this.game.load.image(assetName, 'assets/images/' + assetName + '.png');

           });

     }

});


Использование текстур

private background: Phaser.Sprite;

this.background = this.game.add.sprite(0, 0, Images.Background);
