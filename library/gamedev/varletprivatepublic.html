Переменные (var, let, private, public)

Языки программирования ›› TypeScript ›› Переменные и типы данных ››

Переменная var

var isDone: boolean = false;

var height: number = 6;

var name: string = "bob";

name = 'smith';



Переменная let

Область видимости переменной let – блок {...}

Переменная let видна только после объявления.

При использовании в цикле, для каждой итерации создаётся своя переменная.


let isDone: boolean = false;

let height: number = 6;

let name: string = "bob";

name = 'smith';


Локальные и глобальные переменные

private isDone: boolean = false;

private height: number = 6;

private name: string = "bob";


public isDone: boolean = false;

public height: number = 6;

public name: string = "bob";


Константы

public static GAME_WIDTH: number = 1980;

public static GAME_HEIGHT: number = 1024;
