Работать со слайс изображением (nineSlice)

Подключение плагина в классе Game (файл app.ts)


Phaser.Device.whenReady(() => {

       this.plugins.add(Fabrique.Plugins.NineSlice);     //NineSlice plugin for Phaser!

});


Загрузка изображения


this.game.load.nineSlice(Images.Background, 'assets/images/' + Images.Background + ".png", 20);



Использование слайс изображения


private background:Fabrique.NineSlice;


this.background = this.game.add.nineSlice(0, 0, Images.Background, this.game.width, this.game.height);


this.background.destroy();

this.background = this.game.add.nineSlice(0, 0, Images.Background, this.game.width, this.game.height);

this.background.sendToBack();
