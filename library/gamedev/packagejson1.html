Использование NodeJS

Установка из под package.json

Сборка с помощью Grung

Создайте в корне проекта файл package.json


{

 "name": "sfcards",

 "title": "SFCards",

 "author": "CatfishStudio",

 "version": "1.0.0",

 "description": "This is the game Street Fighter Cards",

 "contributors": [

   {

     "name": "Evgeniy Somov",

     "email": "somov@i.ua"

   }

 ],

 "repository": {

   "type": "git",

   "url": "https://github.com/CatfishStudio/Street-Fighter-Cards"

 },

 "private": true,

 "devDependencies": {

   "phaser-ce": "^2.9.4",

   "typescript": "^2.0.x",

   "node-static": "^0.7.10"

 },

 "engines": {

   "node": "5.x.x"

 }

}



Откройте консоль и введите команду npm install чтобы выполнить установку пакетов указанных в файле package.json
