Отладка проекта F5


Чтобы собрать все ts файлы в один файл необходимо:


нажмить ctrl+p (в редакторе Visual Code) и ввести команду task tsc


Чтобы все созданные или добавленные ts файлы были видны отовсюду их нужно прописать в файле reference.ts


Дополнительно:


Ctrl + Shift + B - выполняет сборку task

F5 - отладка

Ctrl + P - открывает консоль
