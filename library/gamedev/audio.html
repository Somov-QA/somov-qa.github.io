Звуки (audio)

Игровые движки ›› Phaser ›› JavaScript ››

Загрузка звуковых файлов


game.load.audio('sound1', 'assets/sound1.mp3');

game.load.audio('sound2', 'assets/sound2.mp3');

Инициализация звуков


var sound1 = game.add.audio('sound1');

sound1.loop = true;


var sound2 = game.add.audio('sound2');

Воспроизведение и остановка звука


sound1.stop();

sound2.play();
