Blender: установка утилиты

Цвет экрана

1. Устанавливаем Blander version **>= 2.73.0**


2. Копируем папку io_three  из ..\utils\exporters\blender\addons\  в папку с установленным Blander ..\Blender\2.73\scripts\addons


3. Запускаете Blander


4. Переходите в "File" > "User Preferences" > "Addons"


5. В поиске пишете three


6. Ставите флажок Enable an Addon


7. Нажимаете кнопку Save User Settings


Теперь  чтобы им экспортировать модель нужно сделать следующее: "File" > "Export" > "Three.js (.json)"


При выгрузке выберите следую параметры:


Vertices - ok

Faces - ok

Normal - ok

UVs - ok

Apply Modifiers -ok

Face Materials - ok

Embed animation - ok

Materials - ok

Textures - ok

Copy textures - ok (указать путь куда копировать текстуры)

Enable Precision - ok

Indent JSON - ok
