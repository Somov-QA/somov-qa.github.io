Message passing (передача сообщений)

Передача сообщений


msg.post(receiver, message_id[, message])

       receiver - получатель (строка) (string | url | hash | nil)

       message_id - идентификатор сообщения. (строка) (string | hash)

       [message] - таблица данных бля отправки через сообщение.


msg.post(my_url, "my_message", {my_parameter = "my_value"})



Создание URL объекта


msg.url(urlstring)

       urlstring - строка в формате "[socket:][path][#fragment]"

               socket - идентификатор коллекции

               path - идентификатор экземпляра

               fragment - идентификатор компонента


local my_url = msg.url("#my_component")

local my_url = msg.url("my_collection:/my_sub_collection/my_instance#my_component")

local my_url = msg.url("my_socket:")


Конструкция URL


local url = msg.url()

url.socket = "main_collection"

url.path = hash("/my_collection/my_gameobject")

url.fragment = "my_script"

msg.post(url, "my_message")



Использование системных сокетов:  @physics, @render, @system


msg.post("@system:", "toggle_profile")

Отправка сообщения из скрипта одной прокси коллекции в скрипт другой прокси коллекции в пределах глобальной коллекции main


Добавляем компонент игровой объект

id: main_go

в состав входит:

Скрипт id: script


Добавляем компонент игровой объект

id: test_go

в состав входит:

1. Прокси коллекция

   id: test_collection_proxy_1

2. Прокси коллекция

   id: test_collection_proxy_2

Файл: main.script

function init(self)

       msg.post("#", "create_collection_proxy") -- сообщение для текущего скрипта

end


function on_message(self, message_id, message, sender)

       if message_id == hash("create_collection_proxy") then

               msg.post("test_go#test_collection_proxy_1", "load") --загружаем прокси коллекцию

               msg.post("test_go#test_collection_proxy_2", "load") --загружаем прокси коллекцию

               

       elseif message_id == hash("proxy_loaded") and sender.fragment == hash("test_collection_proxy_1") then

               -- если прокси коллекция загружена

               msg.post(sender, "init")                -- инициализация загруженной прокси коллекции

               msg.post(sender, "enable")        -- включение загруженной прокси коллекции

     

       elseif message_id == hash("proxy_loaded") and sender.fragment == hash("test_collection_proxy_2") then

               -- если прокси коллекция загружена

               msg.post(sender, "init")                -- инициализация загруженной прокси коллекции

               msg.post(sender, "enable")        -- включение загруженной прокси коллекции

       end

end
