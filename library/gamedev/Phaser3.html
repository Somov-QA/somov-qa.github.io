Подключение Phaser к проекту

Шаблон проекта

1. Устанавливаем Visual Studio 2015


2. Создаём TypeScript проект в Visual Studio 2015

       Файл -> Создать -> Проект

       Шаблоны -> TypeScript (выбрать "Приложение HTML, разработка с использованием Т...TypeScript" )


3. В папке проекта следующие папки:


       Создаём папку [assets] в которую складываем необходимые ассеты


       Создаём папку [lib] и копируем в неё файлы из архива Phaser

               phaser.min.js        ( из папки: phaser-2.4.6\build\ )

               phaser.d.ts        ( из папки: phaser-2.4.6\typescript\ )

               p2.d.ts

               phaser.comments.d.ts

               pixi.comment.d.ts

               pixi.d.ts


4. Подключаем phaser.d.ts к проекту как ссылку.

       В окне "Обозреватель решений" выделяем проект и нажимаем ctrl+alt+A

       Переходим в папку [lib] и выбираем файл phaser.d.ts

       На кнопке "Добавить" нажимаем стрелку вниз и выбираем пункт "Добавить как связь"
