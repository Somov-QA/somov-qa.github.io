Работа с cookie


Для установления данных в файл cookie используется функция setcookie("имя", "значение")

       name - имя cookie (string)

       value - значение cookie (string)

       expire - срок действия cookie (int)

       path - путь на сервере в пределах которого cookie доступен (string)

       domain - домен, на котором доступен cookie (string)

       secure - указывает на использование cookie только при безопасном соединении. (int)

Для удаления используется та же функция с пустым значением setcookie("имя", "")

---------------------------------------------------------------------------------------

<?php

       setcookie("color", "green");

?>

<html>

<head><title></title></head>

<body>

       <a href="read_cookie.php"></a>

</body>

</html>

---------------------------------------------------------------------------------------

<html>

<head><title></title></head>

<body>

       <?php

               if(isset($_COOKIE['color']) != false)

               {

                       echo "Значение cookie: ", $_COOKIE['color'];

               }else{

                       echo "Cookie не установлено";

               }

       ?>

</body>

</html>

---------------------------------------------------------------------------------------

Cookie можно организовать в массив


setcookie("cookie[1]", "color");

setcookie("cookie[2]", "data");

setcookie("cookie[3]", "time");
