Математические операции


+        сложение

-        вычитание

*        умножение

/        деление

%        остаток от деления двух чисел


Комбинированные операторы присвоения


$var += 21;                // аналогично $var = $var + 21

$var -= 21;                // аналогично $var = $var - 21

$var *= 21;                // аналогично $var = $var * 21

$var /= 21;                // аналогично $var = $var / 21


Инкремент и Декремент


++$var;                // аналогично $var = $var + 1;

--$var;                // аналогично $var = $var - 1;

$var++;                // аналогично $var = $var + 1;

$var--;                // аналогично $var = $var - 1;


Префиксные и постфиксные операции инкремента и декремента


$var = 15;

echo "Значение: " , $var++ , "<br>";        // Значение: 15 (сначала показать, потом инкременация)

echo "Значение: " , ++$var , "<br>";        // Значение: 16 (сначала инкременация, потом показать)
