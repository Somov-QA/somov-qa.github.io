Видимость переменных


$var = 45;        // глобальная переменная


function myfunc()

{

       $var = 100;        // локальная переменная

       echo "<br> Значение внутри функции: ".$var;

}


myfunc();

echo "<br> Значение извне функции: ".$var;


/* В результате получим:

*  Значение внутри функции: 100

*  Значение извне функции: 45

*/



Чтобы переменная была доступна из любого места программы её объявляют глобальной global


$var = 10;

function myfunc()

{

       global $var, $gvar;

       $gvar = $var + 45;

}


myfunc();

echo "Обращение к глобальной переменной: ", $gvar;



Статические переменные static


function mycount()

{

       static $counter;        // статическая переменная значение которой не стирается

       $counter++;

       echo "<br>Значение: ", $counter;

}


for ($i = 0; $i < 10; $i++) mycount();


/*

* Значение: 1

* Значение: 2

* Значение: 3

* Значение: 4

* Значение: n

* Значение: 10

*/
