Суперглобальные переменные


$GLOBAL - массив всех глобальных переменных

$_SERVER - массив параметров работы сервера

$_GET, S_POST - массивы параметров переданные в сценарий

$_COOKIE - массив данных, соханяемых в файл cookie

$_REQUEST - массив введенных пользователем данных (включая $_GET, S_POST, $_COOKIE)

$_FILES - массив данных о загруженном файле

$_ENV - массив переменных окружения

$_SESSION - массив переменных сеанса работы пользователя со сценарием.



echo "Ваш IP: ".$_SERVER['REMOTE_ADDR'];

echo "Вы пришли с сайта: ".$_SERVER['HTTP_REFERER'];
