Загрузка и обработка файлов


форма гля загрузки файлов


<form enctype="multipart/form-data" method="post" action="form_file.php">

       <input name="myfile" type="file"><br>

       <input type="submit" value="Загрузить">

</form>


Сценарий обработки загруженного файла


<?php

       $handle = fopen($_FILES["myfile"]["tmp_name"], "r");

       while (feof($handle) == 0)

       {

               $file_content = fgets($handle);

               echo $file_content, "<br>";

       }

       fclose($handle);

?>
