Работа с функциями


function summa($x, $y, $z = 15)

{

       $result = $x + $y + $z;

       return $result;

}


$var = summa(5, 10);

echo "Результат: ".$var;



Передача аргументов по ссылке &


function myfunc(&$str)

{

       $str = 'значение изменено';

}


$var = 'значение';

myfunc($var);

echo "Результат: ".$var;        // в результате строка "значение изменено"



Функции могут возвращать ссылки как результат своей работы


function &myfunc(&$var)

{

       return $var;

}


$num = 7;

$edit = &myfunc($num);

$edit -= 3;

echo "Результат: ".$num;        // в результате 4
