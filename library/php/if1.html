Оператор if


==        равно

===        твердое равенство

!=        неравенство

!==        твердое неравенство

<        меньше

>        больше

<=        меньше или равно

>=        больше или равно


!                отрицание

&&        AND        логическое И

||        OR        логическое ИЛИ

^        XOR        логическое исключительное ИЛИ


Конструкция условия


if ($var1 > $var2) {

       echo "больше";

} elseif ($var1 < $var2) {

       echo "меньше";

} else {

       echo "равно";

}
