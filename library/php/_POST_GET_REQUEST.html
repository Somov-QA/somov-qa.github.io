Работа с формами


Передача значений типа post и get


<form method="post" action="page.php?code=1&project=game">

       <input name="name" type="text">

       <input type="submit" value="OK">

</form>


Получение значений типа post и get


<?php

       if(isset($_REQUEST["name"]))        // проверка существования

       {

               echo $_POST["name"];

               echo $_GET["code"];

               echo $_GET["project"];

       }

?>
