Счётчик показов


<?php

       session_start();

?>

<html>

<head>

<title>Использование сеансов</title>

</head>

<body>

       <?php

               if(isset($_SESSION['counter']) == false)

               {

                       $_SESSION['counter'] = 0;

               }


               $_SESSION['counter']++;


               echo "Эту страницу открывали: ", $_SESSION['counter'], " раз.";

       ?>

</body>

</html>



Чтобы удалить данные из массива сеанса необходимо воспользоваться функцией unset()


session_start();

unset($_SESSION['counter']);


Чтобы получить или установить идентификатор сеанса, можно использовать функцию session_id()  PHPSESSID
