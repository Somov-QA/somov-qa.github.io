Операторы повторного использования кода


Имеется четыре оператора для включения в сценарий повторно используемых фрагментов кода:

include()                // при сбое, сообщает о нештатной ситуации

       include_once()        // исполняется только один раз

       require()                // при сбое, выдаёт неисправимую ошибку

       require_once()        // исполняется только один раз


<?php

       include("window.php");

?>


<?php

       require("window.php");

?>
