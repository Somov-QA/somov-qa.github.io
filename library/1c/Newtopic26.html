Специальные

Специальные

СоздатьОбъект(<ИмяАгрегатногоТипа>); Создает объект агрегатного типа данных.

СтатусВозврата(<Статус>); Определение или установка статуса возврата предопределенной процедуры.

Возврат;

ТипЗначения(<Значение>); Возвращает тип данных значения в виде числа.

ТипЗначенияСтр(<Значение>); Возвращает строковое обозначение типа данных.

ПустоеЗначение(<Значение>); Функция определяет, является ли пустым переданное в параметре значение.

ПолучитьПустоеЗначение(<Тип>); Возвращает пустое значение заданного типа

НазначитьВид(<Значение>,<Вид>); Процедура НазначитьВид используется для установки значениям типа ''Документ неопределенного вида'', ''Справочник неопределенного вида'', ''Счет неопределенного вида'' конкретного вида. При этом, если существующее значение не соответсвует вновь установленному виду - то оно очищается

ПрефиксАвтоНумерации(<ИмяВида>,<Префикс>); Установить префикс для автоматического создания новых номеров.

ПолучитьЗначенияОтбора(<ИмяОтбора>,<СписокЗначений>,<ДатаНач>,<ДатаКон>); Выбирать все существующие значения отбора

ЗаписьЖурналаРегистрации(<Коммент>,<ТипСобытия>,<Событие>,<Объект>,<Категория>); Выводит строку текста в системный журнал регистрации.

КомандаСистемы(<СтрокаКоманды>);  Вызывает на исполнение команду DOS.

ЗапуститьПриложение(<СтрокаКоманды>);  Выполняет запуск любого приложения. В отличие от функции 'КомандаСистемы' запуск приложения выполняется непосредственно, минуя командный интерпретатор

ЗавершитьРаботуСистемы(<ФлагСохранения>); вызывает завершение рабоы системы.

НайтиПомеченныеНаУдаление(<Объекты>); Находит все помеченные на удаление объекты.

НайтиСсылки(<Объекты>,<Ссылки>); Находит ссылки на объекты, переданные в списке значений.

УдалитьОбъекты(<Объекты>,<Проверять>,<Ссылки>); Удаляет объекты, переданные в списке значений.

ОбработкаОжидания(<ИмяПроцедуры>,<ИнтервалВызова>); Инициирует периодический вызов процедуры глобального модуля с заданным интервалом времени.

ОткрытьФорму(<ИмяАгрегатногоТипа>); Открывает форму агрегатного типа данных. (Например: ОткрытьФорму("Справочник.Товары");

ОткрытьФормуМодально(<Документ>,<КонтекстФормы>,<РежимПросмотра>); Функция ОткрытьФормуМодально позволяет открыть модальную форму документа, справочника, журнала документов и других форм (происходит блокировка оставляя доступным только эту форму)
