Поле со списком

Добавить значение в список:

<ПолеСоСписком>.ДобавитьЗначение("Значение");

<Списком>.ДобавитьЗначение("Значение");


Если список с пометками то их можно установить и получить:

<Списком>.Пометка(Позиция, 1);

и

Если <Списком>.Пометка(Позиция) = 1 Тогда


Получить выбранное значение в поле со списком или списке:

Значение = <ПолеСоСписком>.ПолучитьЗначение(<ПолеСоСписком>.ТекущаяСтрока(),"")

и

Значение = <Списком>.ПолучитьЗначение(н)


Получить размер всего списка:

Для н = 1 По <Списком>.РазмерСписка() Цикл

и

Для н = 1 По <ПолеСоСписком>.РазмерСписка() Цикл
