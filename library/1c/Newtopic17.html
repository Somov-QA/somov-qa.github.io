Модули

Модули

Модуль приложения (НаКлиенте) - предназначен для обработки событий интерактивного запуска и завершения работы, так же перехват внешних событий от оборудования. Не сработает в режиме com-соединения. Модуль выполняется на клиенте.

В данном модуле можно размещать разделы:

- раздел объявления переменных,

- раздел описания произвольных процедур и функций

- раздел основной программы

Не рекомендуется использовать "экспортные" переменные, процедуры и функции так как это замедлит запуск системы.

[Конфигурация] -> контекстное меню [Открыть модуль управляемого приложения]

[Конфигурация] -> контекстное меню [Открыть модуль обычного приложения]


Модуль внешнего соединения (НаСервере) - предназначен для обработки событий запуска и завершения в режиме com-соединения. Модуль компилируется на сервере.

Возможно описывать экспортные переменные и экспортные методы.

[Конфигурация] -> контекстное меню [Открыть модуль внешнего соединения]


Модуль сеанса (НаСервере) - нужен для того, чтобы инициализировать параметры сеанса. Всегда исполняется в привилегированном режиме. Модуль компилируется на сервере.

- только раздел описания процедур и функций

- нет раздела описания переменных

- нет раздела основной программы

- нельзя описывать "экспортные" переменные и методы.

Параметры сеанса — это быстрые глобальные переменные, значения которых доступны в любом месте конфигурации.

[Конфигурация] -> контекстное меню [Открыть модуль сеанса]


Общие модули (НаКлиенте, НаСервере) - описание общих алгоритмов.

- только раздел описания процедур и функций

- описание экспортынх процедуры и функции

- глобальная переменная: параметры сеанса, экспортные переменные модуля приложения.

Директивы компиляции указываются в свойствах модуля.

Глобальный - объявленные в данном модуле экспортные методы будут доступны из вне напрямую.

Вызов сервера - позволяет на клиенте вызвать методы которые имеют серверную директиву.

Привилегированный - предоставляет полные права доступа

[Конфигурация] -> [Общие] -> [Общие Модули]


Модуль формы (НаКлиенте, НаСервере) - обработка действий пользователя.

Каждая процедура и функция должны иметь директиву компиляции.

- раздел объявления переменных,

- раздел описания произвольных процедур и функций

- раздел основной программы

[Конфигурация] -> [Спр, Док, и т.п.] -> [ИмяОбъекта]  -> [Формы]  -> [ИмяФормы]  -> закладка "Модуль"


Модуль Объекта (НаСервере) - предназначен для обработки стандартных событий справочников, документов, планов счетов, и др. Можно выполнять проверку корректности заполнения данных при записи. Модуль компилируется на сервере.

- раздел объявления переменных,

- раздел описания произвольных процедур и функций

- раздел основной программы

- описание "экспортных" переменных и методов. (расширяют класс объекта)

[Конфигурация] -> [Спр, Док, и т.п.] -> [ИмяОбъекта]  -> контекстное меню [Открыть модуль объекта]


Модуль менеджера значений (НаСервере) - предназначен для констант

[Конфигурация] -> [Константы] -> [ИмяКонстанты]  -> контекстное меню [Открыть модуль менеджера значений]


Модуль набора записей (НаСервере) - предназначен для регистра

[Конфигурация] -> [Регистры...] -> [ИмяРегистра]  -> контекстное меню [Открыть модуль набора аписей]


Модуль менеджера (НаКлиенте, НаСервере) - описываются методы для таких объектов как справочники, документы, регистры и пр.

[Конфигурация] -> [Спр, Док, и т.п.] -> [ИмяОбъекта]  -> контекстное меню [Открыть модуль менеджера]
