Временный расчет. (не зависимо от точки актуальности.)

ВременныйРасчет();  - Установить флаг участия регистра во временном расчете

РассчитатьРегистрыНа(<Дата>); - Рассчитать все регистры с установленным флагом временного расчета на начало события

РассчитатьРегистрыПо(<Дата>); - Рассчитать все регистры с установленным флагом временного расчета на конец события


!!! Обязательным является проверка актуальности итогов в момент проведения расходных документов.

Используется только в модуле документа , процедура ОбработкаПроведения()


Если ИтогиАктуальны() = 0 Тогда

// делаем временный расчет.

КонецЕсли;


Пример: Узнаем остатки на начало выбранного периода.

Рег = СоздатьОбъект("Регистр.ОстаткиТоваров");

Рег.ВременныйРасчет(); //Установить флаг участия регистра во временном расчете

Рег.УстановитьЗначениеФильтра("Склад",Склады); // "Склад" - это идентификатор измерения регистра, а Склады - это реквизит диалога с типом Справочник.Склады

РассчитатьРегистрыНа(ДатаНач); // Рассчитать все регистры с установленным флагом временного расчета на начало события

Рег.СводныеОстатки(спрТовар.ТекущийЭлемент()); // Извлечь сводные ресурсы по регистру.

Остаток = Рег.Количество; // Количество - это ресурс регистра.


Чтобы получить остатки на конец выбранного периода нужно заменить только одну строчку:

РассчитатьРегистрыПо(ДатаКон); // Рассчитать все регистры с установленным флагом временного расчета на конец события
