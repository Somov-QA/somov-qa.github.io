Методы оборотных регистров. Получение оборотов.


ИспользоватьПериод(<Дата>) - Установить период выборки итогов оборотного регистра.

ИспользоватьПериод(<Год>,<Месяц>,<День>)

ИспользоватьПериод(<Год>,<НомерНедели>)

ИспользоватьПериод(<Год>,<Месяц>,<НомерДекады>)

ИспользоватьПериод(<Год>,<Месяц>)

ИспользоватьПериод(<Год>,<НомерКвартала>)

ИспользоватьПериод(<Год>)

Итог(<Измерен1>,<Измерен2>...,<ИмяРесурса>) - Возвращает значение итога заданного ресурса по ранее установленному периоду оборотного регистра.

Итоги(<Измерение1>,<Измерение2>...) - Извлечь итоги всех ресурсов оборотного регистра по ранее установленному периоду оборотного регистра.

СводныйИтог(<Измерен1>,<Измерен2>...,<ИмяРесурса>) - Возвращает значение сводного итога заданного ресурса по ранее установленному периоду оборотного регистра.

СводныеИтоги(<Измерение1>,<Измерение2>...) - Извлечь сводные итоги всех ресурсов оборотного регистра по ранее установленному периоду оборотного регистра.

ИтогиПолучить() - Получить все ресурсы по регистру.


Пример: Получение суммы и объема продаж товара по всем контрагентам.

// Задаем период, за который нас интересуют продажи, авгест 2009 года.

Регистр.Продажи.ИспользоватьПериод(2009,8);

Регистр.Продажи.СводныеИтоги(,Товар);

Сообщить("Было продано: " + Регистр.Продажи.Количество + " ед. товара   на сумму: " + Регистр.Продажи.Сумма);


Пример: получим значение итога одного ресурса оборотного регистра.

Регистр.ОборотТовара.ИспользоватьПериод(ДатаДок);

КолВо = Регистр.ОборотТовара.Итог(Товар, Контрагент, "Количество");


Пример: получить все итоги регистров оборотного регистра.

Регистр.ОборотТовара.ИспользоватьПериод(ДатаДок);

Регистр.ОборотТовара.Итоги(Товар, Контрагент);

КолВо = Регистр.ОборотТовара.Количество;
