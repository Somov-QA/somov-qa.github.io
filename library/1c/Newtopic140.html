Виды расчетов и Группы расчетов

При создании "Вида расчета" можно настроить вытеснения…  

и вхождение в группу (но это настраивается только при наличии группы)


При создании "Группы расчетов" определяются виды расчетов которые входят в эту группу.


Пример: Перебор всех Видов расчетов с учетом Группы расчетов

Для Ном=1 По Метаданные.ВидРасчета() Цикл

НазваниеВР = Метаданные.ВидРасчета(Ном);

ВР = ВидРасчета.ПолучитьАтрибут(НазваниеВР);

Если ВР.ВходитВГруппу(ГруппаРасчетов.ГруппаОплПоОкл) = 1 Тогда

// Код - это идентификатов.

// Наименование - это комментарий.

Сообщить("Расчет " + ВР.Код + " (" + ВР.Наименование + ") входит в группу " + ГруппаРасчетов.ГруппаОплПоОкл.Наименование);

КонецЕсли;

КонецЦикла;


Пример: Определим виды расчета которые вытесняются видом расчета "ОплатаПоОкладу"

Для Ном = 1 По Метаданные.ВидРасчета() Цикл

НазваниеВР = Метаданные.ВидРасчета(Ном);

ВР = ВидРасчета.ПолучитьАтрибут(НазваниеВР);

Если ВР.ВытесняетсяВидомРасчета(ВидРасчета.ОплатаПоОкладу) = 1 Тогда //Определяет, вытесняется ли текущий вид расчета заданным видом расчета

Сообщить("Вид расчета (" + ВР.Код + ") вытесняется видом расчета (" + ВидРасчета.ОплатаПоОкладу.Код + ")");

КонецЕсли;

КонецЦикла;


Пример: Получить все виды расчета из Группы расчетов

Группа = ГруппаРасчетов.ГруппаОплПоОкл;

Для Ном = 1 По Группа.Количество() Цикл

ВР = Группа.ПолучитьРасчет(Ном);

Сообщить(ВР.Код);

КонецЦикла;
