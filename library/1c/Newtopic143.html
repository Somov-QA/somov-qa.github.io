Ввод документа на основании


1) В методанных выделите нужный документ, и в контекстном меню выберите "Ввод на основании..."

Теперь выберите документ который: является основанием или вводится на основании.


2) В модуле формы документа который будет вводится на основании должна быть предопределенная процедура.

Процедура ВводНаОсновании(ДокументОснование)

// заполнение реквизитов шапки документа

ДатаДок = ДокументОснование.ДатаДок;

…………..

// заполнение табличной части документа.

ДокументОснование.ВыбратьСтроки();

Пока ДокументОснование.ПолучитьСтроку()=1 Цикл

НоваяСтрока();

…………..

КонецЦикла;

КонецПроцедуры


ДокументОснование - содержит весь контекст документа который стал основанием для создания текущего документа.
