Метаданные


1) Доступ к скрытым реквизитам.


Тип = Метаданные.Справочник("Товары").Реквизит("СтавкаНДС").Тип;

Длина = Метаданные.Справочник("Товары").Реквизит("СтавкаНДС").Длина;

Точность = Метаданные.Справочник("Товары").Реквизит("СтавкаНДС").Точность;

или

Тип = Метаданные.Справочник("Товары").Реквизит(номРекв).Тип;

Длина = Метаданные.Справочник("Товары").Реквизит(номРекв).Длина;

Точность = Метаданные.Справочник("Товары").Реквизит(номРекв).Точность;

СтрРекв = Метаданные.Справочник(Строка(СправочникДляВыгрузки.ПолучитьЗначение(СправочникДляВыгрузки.ТекущаяСтрока(),""))).Реквизит(ном).Идентификатор;

Периодич = Метаданные.Справочник(Строка(СправочникДляВыгрузки.ПолучитьЗначение(СправочникДляВыгрузки.ТекущаяСтрока(),""))).Реквизит(ном).Периодический; //0- не периодический , 1- периодический реквизит



2) Получим все справочники, документы, перечисления (их идентификаторы) созданные в конфигурации.

// Все полученные данные записываются в список значений

// Загружаем список всех справочников.

Объект = Метаданные.Справочник();

Для нСпр = 1 По Объект Цикл

СправочникДляВыгрузки.ДобавитьЗначение(Строка(Метаданные.Справочник(нСпр).Идентификатор));

ВставитьТип.ДобавитьЗначение("Справочник." + Строка(Метаданные.Справочник(нСпр).Идентификатор));

СправочникВладелец.ДобавитьЗначение(Строка(Метаданные.Справочник(нСпр).Идентификатор));

КонецЦикла;


Объект = Метаданные.Документ();

Для нСпр = 1 По Объект Цикл

ВставитьТип.ДобавитьЗначение("Документ." + Строка(Метаданные.Документ(нСпр).Идентификатор));

КонецЦикла;


Объект = Метаданные.Перечисление();

Для нСпр = 1 По Объект Цикл

ВставитьТип.ДобавитьЗначение("Перечисление." + Строка(Метаданные.Перечисление(нСпр).Идентификатор));

КонецЦикла;



3) Определим установлен ли флажок "Контроль уникальности" в справочнике.

Если Метаданные.Справочник("Сотрудники").КонтрольУникальности = 1 Тогда

Сообщить("Справочник  [Сотрудники]  для реквизита КОД установлен КОНТРОЛЬ УНИКАЛЬНОСТИ.");

КонецЕсли;

или

Если Метаданные.Справочник(Строка(СправочникДляВыгрузки.ПолучитьЗначение(СправочникДляВыгрузки.ТекущаяСтрока(),""))).КонтрольУникальности = 1 Тогда

Сообщить("Справочник  [" + СправочникДляВыгрузки.ПолучитьЗначение(СправочникДляВыгрузки.ТекущаяСтрока(),"") + "]  для реквизита КОД установлен КОНТРОЛЬ УНИКАЛЬНОСТИ.");

КонецЕсли;
