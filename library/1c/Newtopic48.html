Получить данные из реквизитов табличной части документа


ДокПремия = СоздатьОбъект("Документ.Премия"); //объект агр. типа данных Документ

ДокПремия.ВыбратьДокументы(НачМесяца(ДатаДок), КонМесяца(ДатаДок)); //выбираем документ "Премия" из диапазона дат, для получения номера этого документа.

Если ДокПремия.НайтиПоНомеру(ДокПремия.НомерДок, ДатаДок) = 1 Тогда //находим документ по номеру.

КолСтрокТабличнойЧастиДок = ДокПремия.КоличествоСтрок(); //определяем количество строк табличной части данного документа "Премия"

Для н = 1 По КолСтрокТабличнойЧастиДок Цикл //читаем строки табличной части документа

ДокПремия.ПолучитьСтроку(); //получить следующую строку табличной части

Сотрудник = ДокПремия.Сотрудник; //получить значение из реквизита табличной части. (Сотрудники)

Если НаименованиеФИО = Сотрудник Тогда //если "ФИО" совпадает со значением табличной части документа "Премия" Сотрудник тогда...

Премия = ДокПремия.РазмерПремии; //получаем значение размер премия.

КонецЕсли;

КонецЦикла;

КонецЕсли;
