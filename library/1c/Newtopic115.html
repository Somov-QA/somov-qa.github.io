Работа в режиме запроса


Перед выполнением запроса устанавливаются фильтры по субконто и корреспондирующим субконто.

ИспользоватьСубконто(<ВидСубконто>,<Субконто>,<ТипФильтра>,<ПоГруппам>)

ИспользоватьКорСубконто(<ВидСубконто>,<Субконто>,<ТипФильтра>,<ПоГруппам>)

Пояснение: (<ТипФильтра> если 1-расчет по этому Виду субконто, 2-расчет по значению субконто, 3-субконто не учитывается)


Опции(<ВклЗабалансСуммы>,<ВклОборотСубкСуммы>); // дополнительные настройки.


ВыполнитьЗапрос(<НачалоПериода>,<КонецПериода>,<Счет>,<КоррСчет>,<Валюта>,<ТипИтогов>,<Периодичность>,<ТипСуммы>)

Пояснение: (<ТипСуммы> - принимает значение: 1("С")-сумма, 2("В")-валютная сумма, 3("К")-количество)

Пояснение: (<ТипИтогов> - если 1-остатки и обороты по счетам, 2-обороты между счетами, 3-и то и другое)

Пояснение: (<Периодичность> - может принимать следующие значения:

"Период"        промежуточные итоги не расчитываются

"Операция"        промежуточные итоги расчитываются по операциям.

"Проводка"        по проводкам

"День"                по дная

"Неделя"        по неделям

"Декада"        по декадам

"Месяц"        по месяцам

"Квартал"        по кварталам

"Год"                по годам )



ВключатьСубсчета(<ФлагСчета>,<ФлагКоррСчета>) // Устанавливает режим отбора итогов по субсчетам
