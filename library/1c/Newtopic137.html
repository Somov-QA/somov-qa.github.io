Последовательности документов.


Групповое перепроведение документов можно выполнить в монопольном режиме

в меню Операции -> Проведение документов.

(тут же на закладке последовательности можно узнать границу последовательности) эта закладка отобразится если в конфигурации есть хоть одна последовательность документа.


Создание последовательности досументов.

Последовательность документов - это объект метаданных, который используется совместно с

компонентами "Бухгалтерский учет" и "Оперативный учет" для отслеживания проведения документов в хронологическом порядке.


В разделе метаданных Документы использовать подраздел Последовательность для создания новой последовательности.


Методы:

ПолучитьПозицию() - Получить позицию Границы Последовательности

Получить() - Получить строку описания Границы Последовательности.

ПолучитьДокумент() - Получить документ Границы Последовательности

ПолучитьДату() - Получить дату Границы Последовательности.

ПолучитьВремя(<Часы>,<Минуты>,<Секунды>) - Прочитать время Границы Последовательности.

Установить(<ПоложениеГП>) - Установить ГП (границу последовательности) на новую дату, документ или позицию.

Сравнить(<Докум>) - Сравнить ГП (границу последовательности) с датой, документом или позицией.

ПринадлежитПоследовательности(<Докум>) - Определить, принадлежит ли последовательности заданный документ или вид, заданный строкой.

Проверить(<Докум>) - Проверяет, является ли последовательность непрерывной от ГП (границы последовательности) до заданного документа.


Методы глобального атрибута:

ПолучитьАтрибут(<ИмяПоследовательности>) - Получить значение последовательности по имени. Возвращает значение последовательности.


Пример: Получить границы последовательности.

Сообщить("Граница последовательности: " + Последовательность.Основная.Получить() + Последовательность.Основная.ПолучитьДокумент());

Сообщить("Позиция границы последовательности: " + Последовательность.Основная.ПолучитьПозицию());


Пример: Установка границы последовательности на текущую дату.

Последовательность.Основная.Установить(ТекущаяДата());


Пример: Восстановление последовательности.

Послед = Последовательность.Основная;

Док = СоздатьОбъект("Документ");

Док.ВыбратьДокументы(Послед.ПолучитьПозицию(),);
