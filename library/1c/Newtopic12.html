Список значений

Представляет собой список значений


СписокЗнач = Новый СписокЗначений;

СписокЗнач.Добавить("Понедельник");

СписокЗнач.Добавить("Вторник");

СписокЗнач.Добавить("Среда");


Для Каждого Элемент Из СписокЗнач Цикл

Сообщить(Элемент);        

КонецЦикла;


НайденоеЗначение = СписокЗнач.НайтиПоЗначению("Среда");

Сообщить(НайденоеЗначение.Пометка);


       Показать модельное окно с выбором


Значение = Строка(СписокЗнач.ВыбратьЭлемент("День недели", НайденоеЗначение));


       Показать можельное окно с отметками


СписокЗнач.ОтметитьЭлементы();


Загрузить список значений в список значений расположенный на форме в качестве реквизита.


СписокЗначенийНаФорме = СписокЗнач;
