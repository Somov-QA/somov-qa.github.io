Общие методы регистров.


Пример: отобразить в строке состояния вид регистра.

Состояние (Рег.Вид());


Пример: отобразим пользовательское представление вида в строке состояния.

Состояние (Рег.ПредставлениеВида());


Пример: назначим тип реквизиту регистра тип которого "Неопределенный"

Регистр.ОборотТовара.НазначитьТип("Товар", "Справочник.Товары");


Пример: установим значение атрибута по имени идентификатора.

Рег.УстановитьАтрибут("Сумма", СуммаТов);


Пример: получим значение атрибута по идентификатору.

СуммаТов = Рег.ПолучитьАтрибут("Сумма");


Пример: выберим все движения регистра в указанном периоде.

Рег.ВыбратьДвижения(ДатаНач,ДатаКон); // применил в примере получения прихода и расхода.


Пример: произведем выбор всех движений регистра по указанному документу.

Рег.ВыбратьДвиженияДокумента(Док.ТекущийДокумент());

Пока Рег.ПолучитьДвижение() > 0 Цикл

КонецЦикла;


Пример: установим обратный порядок выборки движения документов.

Рег.ОбратныйПорядок(1);

Рег.ВыбратьДвиженияДокумента(Док.ТекущийДокумент()); // или  Рег.ВыбратьДвижения(ДатаНач, ДатаКон);

Пока Рег.ПолучитьДвижение() > 0 Цикл

КонецЦикла;


Пример: выбираем следующее движение регистра (применяется в цикле)

Пока Рег.ПолучитьДвижение() > 0 Цикл


Пример: получим значение текущего документа, который задал движение регистра.

Док = Рег.ТекущийДокумент();


Пример: получим значение номера строки документа, которая задала движение регистра.

(если в модуле документа перед движением использовался метод ПривязыватьСтроку())

НомСтр = Рег.НомерСтроки();


Пример: произведем перебор текущих или временых остатков по регистру.

(при этом может использоваться фильтр УстановитьФильтр())

Рег.УстановитьФильтр(Клиент, );

Рег.ВыбратьИтоги();


Пример: выбираем очередной остаток по регистру.

Рег.ВыбратьИтоги();

Пока Рег.ПолучитьИтог() = 1 Цикл

КонецЦикла;


Пример: выгружаем все итоги регистра с текущим фильтром.

Рег.УстановитьФильтр(Клиент, );

Рег.ВыгрузитьИтоги(СписокЗнач, 1);


Пример: устанавливаем флаг участия регистра во временном расчете.

Рег.ВременныйРасчет();


Пример: установим фильтр для оптимизации действий с регистром.

Рег.УстановитьФильтр(Клиент, );


Пример: установим значение фильтра для оптимизации расчетов.

Рег.УстановитьЗначениеФмльтра("Товар", ВыбГруппаТоваров, 2);
