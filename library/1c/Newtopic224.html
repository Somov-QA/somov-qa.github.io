НачислениеЗарплаты

Данный документ выполняет начисление зарплаты с учетом прогулов и премий



Данные:

Реквилиты: Нет


Табличные части:


ТЧОсновныеНачисления

Сотрудник (СправочникСсылка.Сотрудники)

ВидыРасчета (ПланВидовРасчетаСсылка.ОсновныеНачисления)

ДатаНачала (Дата)

ДатаОкончания (Дата)

Размер (Число 15, 2)


ТЧДополнительныеНачисления

Сотрудник (СправочникСсылка.Сотрудники)

ВидРасчета (ПланВидовРасчетаСсылка.ДополнительныеНачисления)

ПроцентПремии (Число 10, 2)

БазовыйПериодНачало (Дата)

БазовыйПериодКонец (Дата)



Движение:

НачислениеЗарплаты

НачислениеПремий


Формы:

ФормаДокумента (по умолчанию)


Модуль объекта:


Процедура ОбработкаПроведения(Отказ, Режим)

       

       ПериодРегистрации = НачалоМесяца(Дата);


       // РЕГИСТР: НачислениеЗарплаты

       Движения.НачислениеЗарплаты.Записывать = Истина;

       Движения.НачислениеЗарплаты.Очистить();

       

       Для Каждого ТекСтрокаТЧОсновныеНачисления Из ТЧОсновныеНачисления Цикл

               Движение = Движения.НачислениеЗарплаты.Добавить();

               Движение.Сторно = Ложь;

               Движение.ВидРасчета = ТекСтрокаТЧОсновныеНачисления.ВидРасчета;

               Движение.ПериодДействияНачало = ТекСтрокаТЧОсновныеНачисления.ДатаНачала;

               Движение.ПериодДействияКонец = КонецДня(ТекСтрокаТЧОсновныеНачисления.ДатаОкончания);

               Движение.ПериодРегистрации = ПериодРегистрации;

               Движение.Сотрудник = ТекСтрокаТЧОсновныеНачисления.Сотрудник;

               //Движение.Сумма = 0; // будет рассчитана ниже

               Движение.Оклад = ТекСтрокаТЧОсновныеНачисления.Размер;

       КонецЦикла;

       

       // РЕГИСТР: НачислениеПремий

       Движения.НачислениеПремий.Записывать = Истина;

       Движения.НачислениеПремий.Очистить();

       

       Для Каждого ТекСтрокаТЧДополнительныеНачисления Из ТЧДополнительныеНачисления Цикл

               Движение = Движения.НачислениеПремий.Добавить();

               Движение.Сторно = Ложь;

               Движение.ВидРасчета = ТекСтрокаТЧДополнительныеНачисления.ВидРасчета;

               Движение.БазовыйПериодНачало = ТекСтрокаТЧДополнительныеНачисления.БазовыйПериодНачало;

               Движение.БазовыйПериодКонец = КонецДня(ТекСтрокаТЧДополнительныеНачисления.БазовыйПериодКонец);

               Движение.ПериодРегистрации = ПериодРегистрации;

               Движение.Сотрудник = ТекСтрокаТЧДополнительныеНачисления.Сотрудник;

               Движение.ПроцентПремии = ТекСтрокаТЧДополнительныеНачисления.ПроцентПремии;

       КонецЦикла;

       

       

       Движения.Записать(); // Записываем движение

       

       // Рассчитать оклады с учетом прогулов

       РасчетЗП.РассчитатьОклад(Ссылка, Движения.НачислениеЗарплаты);

       

       // Рассичать премии

       РасчетЗП.РассчитатьПремию(Ссылка, Движения.НачислениеПремий);

       

       // Перерасчет

       РасчетЗП.Перерасчет();
