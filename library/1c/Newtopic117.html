Для обращения к результатам бухгалтерского запроса используются такие атрибуты и функции


БИ.Счет - текущий счет запроса

БИ.Субконто(<ВидСубконто>) - текущее субконто запроса.

БИ.КорСчет - текущий кореспондирующий счет

БИ.КорСубконто(<ВидСубконто>) - текущее корреспондирующее субконто запроса.

БИ.Валюта - текущая валюта запроса.

БИ.НачДата - начальная дата текущего периода.

БИ.КонДата - конечная дата текущего периода.

БИ.Операция - текушая операция запроса.


Сальдо:

БИ.СНД(<типСумма>) - сальдо начальное дебетовое.

БИ.СКД(<типСумма>) - сальдо конечное дебетовое.

БИ.СНК(<типСумма>) - сальдо начальное кредитовое.

БИ.СКК(<типСумма>) - сальдо конечное кредитовое.


Обороты:

БИ.ДО(<типСумма>) - дебетовый оборот по текущему счету, текущему субконто.

БИ.КО(<типСумма>) - кредитовый оборот по текущему счету, текущему субконто.

БИ.КорДО(<типСумма>) - дебетовый оборот по текущему кор. счету, текущему кор. субконто.

БИ.КорКО(<типСумма>) - кредитовый оборот по текущему кор. счету, текущему кор. субконто.


Развернутое сальдо по субконто

СНДРС(<Счет>,<ТипСуммы>,<Валюта>,<Субконто1>,<ТипФильтра1>,<Субконто2>,<ТипФильтра2>)

СНКРС(<Счет>,<ТипСуммы>,<Валюта>,<Субконто1>,<ТипФильтра1>,<Субконто2>,<ТипФильтра2>)

СКДРС(<Счет>,<ТипСуммы>,<Валюта>,<Субконто1>,<ТипФильтра1>,<Субконто2>,<ТипФильтра2>)

СККРС(<Счет>,<ТипСуммы>,<Валюта>,<Субконто1>,<ТипФильтра1>,<Субконто2>,<ТипФильтра2>)


Развернутое сальдо по субсчетам

СНДР(<Счет>,<ТипСуммы>,<Валюта>)

СНКР(<Счет>,<ТипСуммы>,<Валюта>)

СКДР(<Счет>,<ТипСуммы>,<Валюта>)

СККР(<Счет>,<ТипСуммы>,<Валюта>)


Методы.

Субконто(<Номер>|<ВидСубконто>) //Возвращает значение субконто, соответствующее текущему итогу.

КорСубконто(<Номер>|<ВидСубконто>) //Возвращает значение корреспондирующего субконто, соответствующее текущему итогу.

ПредставлениеСубконто(<Номер>|<ВидСубконто>,<Краткое>) //Возвращает cтроковое представление для субконто, соответствующего текущему итогу.

ПредставлениеКорСубконто(<Номер>|<ВидСубконто>,<Краткое>) //Возвращает cтроковое представление для корреспондирующего субконто, соответствующего текущему итогу

ЭтоГруппа() //Определяет, является ли текущее значение группироки запроса по субконто группой или нет. Имеет смысл только для субконто типа ''справочник''.

ВыбранаПоДт() //Определяет, выбран ли итог по дебету.

ВыбранаПоКт() //Определяет, выбран ли итог по кредиту
