Создание формы выбора с передача данных в неё


В форме документа на раквизите создать событие НачалоВыбора чтобы вызвать форму выбора элемента справочника

У документа есть табличная часть в которой находится "ПоставщикиИПрайсы" список прайсов которые необходимо передать в форму.


&НаКлиенте

Функция ПереченьПрайсов()

       СЗПрайсы = Новый СписокЗначений;

       Для Каждого СтрокаТабличнойЧасти Из Объект.ПоставщикиИПрайсы Цикл

               СЗПрайсы.Добавить(СтрокаТабличнойЧасти.Прайс);

       КонецЦикла;

       Возврат СЗПрайсы;

&НаКлиенте

Процедура ПланЗакупкиТоварНачалоВыбора(Элемент, ДанныеВыбора, СтандартнаяОбработка)

       СтандартнаяОбработка = Ложь;

       СЗПрайсы = ПереченьПрайсов();

       

       // Выбранная номенклатура

       СтрокаТабличнойЧасти = Элементы.ПланЗакупки.ТекущиеДанные;

       ВыбраннаяНоменклатура = СтрокаТабличнойЧасти.Номенклатура;

       

       // Прайс лист (по умолчанию)

       //СтрокаТаблицыЗначений  = Объект.ПоставщикиИПрайсы.Получить(0);

       //ПрайсЛисты = СтрокаТаблицыЗначений.Прайс;

       

       // Подготовка параметров

       //ПараметрыДляФормы = Новый Структура("ПрайсЛисты", ПрайсЛисты);

       ПараметрыДляФормы = Новый Структура("СписокЗначенийПрайсы", СЗПрайсы);

       ПараметрыДляФормы.Вставить("ВыбраннаяНоменклатура", ВыбраннаяНоменклатура);

       

       // Открытие формы выбора

       ФормаВыбора = ПолучитьФорму("Справочник.Товары.ФормаВыбора", ПараметрыДляФормы);

       ФормаВыбора.ВладелецФормы = Элементы.ПланЗакупкиТовар;

       ФормаВыбора.Открыть();

Чтобы в форме выбораприменить полученные данные нужно:

1) Открыть форму выбора, выделить в окне реквизиты "Список" и перейти в его свойства.

2) В свойствах устанавливаем флаг на: ПроизвольныйЗапрос

3) Открываем настройки списка
