Журнал расчетов


1) Журнал расчета - это средство для хранения записей расчета.

(для создания журнала расчета необходим справочник.)


Пример: Определение и изменение периода журнала расчетов

ЖР = СоздатьОбъект("ЖурналРасчетов.Зарплата");

Сообщить("Текущий период: " + ЖР.НачалоТекущегоПериода() + " - " + ЖР.КонецТекущегоПериода());

НовыйПериод = ЖР.ПериодПоДате(ДатаНач);

// при смене периода не выполняем системные действия. (0)

Если ЖР.УстановитьТекущийПериод(НовыйПериод, 0) = 1 Тогда

Сообщить("Текущий период: " + ЖР.НачалоТекущегоПериода() + " - " + ЖР.КонецТекущегоПериода());

КонецЕсли;


Пример: Ввод записи в журнал расчетов. (процедура ВвестиРасчет() доступна только в "модуле документа")

Процедура ОбработкаПроведения()

ЖР = СоздатьОбъект("ЖурналРасчетов.Зарплата");

ВыбратьСтроки();

Пока ПолучитьСтроку() = 1 Цикл

ЖР.ВвестиРасчет(ФИО, ВидРасчета.ОплатаПоОкладу, ДатаНач, ДатаКон, Оклад);    

КонецЦикла;

КонецПроцедуры


Пример: Еще один способ добавитьзапись в журнал расчетов.

ЖР = СоздатьОбъект("ЖурналРасчетов.Зарплата");

ВыбратьСтроки();

Пока ПолучитьСтроку() = 1 Цикл

ЖР.Новая();                  

// перечень атрибутов.

ЖР.УстановитьРеквизит("Объект", ФИО);

ЖР.УстановитьРеквизит("ВидРасч", ВидРасчета.ОплатаПоОкладу);

ЖР.УстановитьРеквизит("Документ", ТекущийДокумент());

ЖР.УстановитьРеквизит("РодительскийДокумент", ТекущийДокумент());

ЖР.УстановитьРеквизит("ДатаНачала", ДатаНач);

ЖР.УстановитьРеквизит("ДатаОкончания", ДатаКон);

ЖР.УстановитьРеквизит("Результат", Оклад);

ЖР.УстановитьРеквизит("Фиксирована", 1);

ЖР.УстановитьРеквизит("Рассчитана", 1);

ЖР.Записать();

КонецЦикла;


Пример: Проведение расчета. (реализуется с помощью предопределенной процедуры ПровестиРасчет() в модуле "ВидРасчета")

Процедура ПровестиРасчет()

// Процедура выполняется при проведении расчета

Норма = Календари.Пятидневка.Дней(НачМесяца(ДатаНачала), КонМесяца(ДатаОкончания));

Дни = Календари.Пятидневка.Дней(ДатаНачала, ДатаОкончания);

Часы = Календари.Пятидневка.Часов(ДатаНачала, ДатаОкончания);

Если Норма <> 0 тогда

// Величину оклада берем из документа.

Документ.ВыбратьСтроки();

Пока Документ.ПолучитьСтроку() = 1 Цикл

Результ = Результ + Дни / Норма * Документ.Оклад;

Результат = Результ;

КонецЦикла;

КонецЕсли;

КонецПроцедуры


Запустить проведение расчета текущей записи можно двумя способами:

1. В ручную (в журнале расчетов нажать кнопку "стрелочка" - расчитать запись.)

2. Программно с помощью функции ВыполнитьРасчет()

ЖР.ВыбратьПериод(ЖР.НачалоТекущегоПериода());

Пока ЖР.ПолучитьЗапись() = 1 Цикл

Если ЖР.Документ = ТекущийДокумент() Тогда

ЖР.ВыполнитьРасчет();

КонецЕсли;

КонецЦикла;


Пример: Получим сумму всех начислений по сотруднику.

ЖР = СоздатьОбъект("ЖурналРасчетов.Зарплата");

ЖР.ВыбратьЗаписи(ДатаНач, ДатаКон);

Сум = 0;

Пока ЖР.ПолучитьЗапись() = 1 Цикл

Если ЖР.Объект <> Сотрудник Тогда

Продолжить;

КонецЕсли;

Если ЖР.ВидРасч.ВходитВГруппу(ГруппаРасчетов.ГруппаОплПоОкл) = 1 Тогда

Сум = Сум + ЖР.Результат;

КонецЕсли;

КонецЦикла;

Сообщить("Сумма всех начислений сотрудника " + Сотрудник + " составила:  " + Сум);


2) Использование ЗАПРОСА при работе с записями журнала расчетов.

Пример: Получим сумму всех начислений по сотруднику. (с использованием запроса.)


Запрос = СоздатьОбъект("Запрос");

ТекстЗапроса = "//{{ЗАПРОС(ЗапросПоНачислениям)

|Период С ДатаНач По ДатаКон;

|Объект = ЖурналРасчетов.Зарплата.Объект;

|ВидРасч = ЖурналРасчетов.Зарплата.ВидРасч;

|Результат = ЖурналРасчетов.Зарплата.Результат;

|Функция Сум = Сумма(Результат);

|Группировка Объект;

|Условие(ВидРасч.ВходитВГруппу(ГруппаРасчетов.ГруппаОплПоОкл)=1);

|"; //}}ЗАПРОС

Если Запрос.выполнить(ТекстЗапроса) = 0 Тогда

Возврат;

КонецЕсли;

Пока Запрос.Группировка(1) = 1 Цикл

Сообщить("Сумма всех начислений сотрудника " + Сотрудник + " составила:  " + Запрос.Сум);

КонецЦикла;
