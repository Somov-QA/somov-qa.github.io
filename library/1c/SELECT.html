SELECT - Инструкция

SELECT ALL/DISTINCT элементы

FROM таблицы

WHERE условие отбора

GROUP BY столбец для группирования

HAVING условие отбора

ORDER BY условие сортировки


Выбрать все строки из таблицы


SELECT * FROM table1;


Вычисляемые столбцы. (в столбцах можно производить вычисления.)


SELECT (elem1 – elem2) FROM table1;


Использование констант. (текстовая константа.)


SELECT elem1, 'my text', elem2 FROM table1;
