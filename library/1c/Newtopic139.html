Календарь

1) При создании календаря настраиваются поля:

Стартовая дата -  любой понедельник.

Длина дня - состоит как правило из 7-ми значений (7 дней) например:

8.00        восемь часов, понедельник.

8.00        восемь часов, вторник.

8.00        восемь часов, среда.

8.00        восемь часов, четверг.

8.00        восемь часов, пятница

0.00        восемь часов, субота (выходной)

0.00        восемь часов, воскресение (выходной)


2) Чтобы выполнить автозаполнение календаря, нужно в 1С:Предприятие7.7 , в меню Операции -> Календарь (кнопка Автозаполнение)

Чтобы произвести ручное заполнение календаря нужно тутже выбрать день и ввести количество рабочих дней.


3) Ввести празнечные дни можно в  меню Операции -> Праздники и задать "особые" дни.

тут указывается дата празднечного для , и количество рабочих часов в этот день.


Но тогда в автозаполнении календаря нужно установить флажок Учитывать праздники (эти праздники тоже будут выделены красным)

Пример: Заполнение календаря программным способом.

Календарь = Календари.Пятидневка;

Календарь.Автозаполнение(ДатаНач, ДатаКон); //Выполняет автозаполнение календаря в заданном периоде

Календарь.ВыбратьДаты(ДатаНач, ДатаКон); //Открывает выборку записей календаря.

Пока Календарь.СледующаяДата() = 1 Цикл

ТекДата = Календарь.Дата; // получаем дату текущей строки календаря

ДеньНедели = НомерДняНедели(ТекДата);   //Возвращает порядковый номер дня недели для заданной даты

Календарь.УчитыватьПраздники(1);        //Учитывать праздники которые введены нами

Если (ДеньНедели = 6) или (ДеньНедели = 7) Тогда

Календарь.УстановитьАтрибут("Значение", 0); //Установить значение атрибута по имени идентификатора

Иначе                                                                             // атрибуты (Дата - только чтение) и (Значение)

Календарь.УстановитьАтрибут("Значение", 8);

КонецЕсли;

КонецЦикла;


Пример: получение данных из календаря. (Дней, Часов)

Сообщить("Количество рабочих дней по календарю пятидневной рабочей недели за " + ДатаМесяц(ДатаНач) + " месяц. :" + Календари.Пятидневка.Дней(ДатаНач, ДатаКон));

Сообщить("Количество рабочих часов по календарю пятидневной рабочей недели за " + ДатаМесяц(ДатаНач) + " месяц. :" + Календари.Пятидневка.Часов(ДатаНач, ДатаКон));


Пример: работа с объектом "Праздники"

Пр = СоздатьОбъект("Праздники");

Пр.Новый('23.02.2009',0);

Пр.Новый('07.03.2009',4);

Пр.Новый('08.03.2009',0);

Пр.ВыбратьДаты(ДатаНач, ДатаКон);

Пока Пр.СледующаяДата() = 1 Цикл

Если Пр.Значение = 0 Тогда

Сообщить("Праздник " + Пр.Дата);

Иначе

Сообщить("Предпразничный день " + Пр.Дата);

КонецЕсли;

КонецЦикла;
