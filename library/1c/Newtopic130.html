Запись движения по регистру.

(!!! эти методы доступна только в модуле документа !!!)

!!!   Для того чтобы документ был доступен для регистра нужно: установить флажок Оперативный учет в настройках документа или сразу при создании.


ДвижениеПриходВыполнить() - для записи прихода по регистру остатков.

ДвижениеРасходВыполнить() - для записи расхода по регистру остатков.

ДвижениеВыполнить() - для записи движения по регистру оборотов.


ПривязыватьСтроку(<НомерСтроки>) - Записать номер строки документа в движении регистра

ДвижениеПриход(<Измерение1>,<Измерение2>...,<Ресурс1>,<Ресурc2>...) - Выполнить запись прихода в регистр по параметрам.

ДвижениеРасход(<Измерение1>,<Измерение2>...,<Ресурс1>,<Ресурc2>...) - Выполнить запись расхода в регистр по параметрам.

Движение(<Измерение1>,<Измерение2>...,<Ресурс1>,<Ресурc2>...) - Выполнить запись движения в оборотный регистр по параметрам.


Пример: записать ПРИХОД.

Процедура ОбработкаПроведения()

ВыбратьСтроки();

Пока ПолучитьСтроку() = 1 Цикл

//Запись движения по РЕГИСТРУ ------------------------------

Регистр.ОстаткиТоваров.Товар = Товар;

Регистр.ОстаткиТоваров.Склад = Склад;

Регистр.ОстаткиТоваров.Количество = Количество;

Регистр.ОстаткиТоваров.Сумма = Сумма;

Регистр.ОстаткиТоваров.ДвижениеПриходВыполнить();

КонецЦикла;

КонецПроцедуры


Пример: записать РАСХОД.

Процедура ОбработкаПроведения()

ВыбратьСтроки();

Пока ПолучитьСтроку() = 1 Цикл

//Запись движения по РЕГИСТРУ ------------------------------

Регистр.ОстаткиТоваров.Товар = Товар;

Регистр.ОстаткиТоваров.Склад = Склад;

Регистр.ОстаткиТоваров.Сумма = Сумма;

Регистр.ОстаткиТоваров.ДвижениеРасходВыполнить();

КонецЦикла;

КонецПроцедуры


Пример: процедура расходной накладной выполняется с использованием проверки остатков в момент проведения на случай если мы производим востановление последовательности документов (после ввода документа задним числом)

Процедура ОбработкаПроведения()

//Проверка остатков на случай перепроведения-----------------------

Рег = СоздатьОбъект("Регистр.ОстаткиТоваров");

СписокТоваров = СоздатьОбъект("СписокЗначений");

ВыгрузитьТабличнуюЧасть(СписокТоваров, "Товар");

Рег.УстановитьЗначениеФильтра("Товар", СписокТоваров,2); // 2 - значит искать вхождения

Рег.УстановитьЗначениеФильтра("Склад", Склад);

Если ИтогиАктуальны() = 0 Тогда

// делаем временный расчет

Рег.ВременныйРасчет();

РассчитатьРегистрыНа(ТекущийДокумент());

КонецЕсли;

//-----------------------------------------------------------------


ВыбратьСтроки();

Пока ПолучитьСтроку() = 1 Цикл

//Запись движения по РЕГИСТРУ оборот ------------------------------

Регистр.ОборотТовара.Товар = Товар;

Регистр.ОборотТовара.Контрагенты = Контрагенты;

Регистр.ОборотТовара.Количество = Количество;

Регистр.ОборотТовара.Сумма = Сумма;

Регистр.ОборотТовара.ДвижениеРасходВыполнить();

// расчитаем остатки по текущему товару

Рег.Остатки(Товар, Склад);

ОстКол = Рег.Количество;

ОстСум = Рег.Сумма;

Если ОстКол > Количество тогда

// Расчитать среднюю себестоимость

Себнстоимость = ОстСум / ОстКол * Количество;

ИначеЕсли ОстКол = Количество Тогда

//Если списываем последнюю единицу товара,

//то, чтобы избавится от ошибок округления,

//за себестоимость берем всю оставшуюся сумму.

Себестоимость = ОстСум;

Иначе

Сообщить("В наличии " + ОстКол + " ед. товара " + Товар + " на складе: " + Склад + " из требуемых " + Количество);

НеПроводитьДокумент();

Возврат;

КонецЕсли;


//Запись движения по РЕГИСТРУ расход ------------------------------

Регистр.ОстаткиТоваров.Товар = Товар;

Регистр.ОстаткиТоваров.Склад = Склад;

Регистр.ОстаткиТоваров.Количество = Количество;

Регистр.ОстаткиТоваров.Сумма = Сумма;

Регистр.ОстаткиТоваров.ДвижениеРасходВыполнить();

КонецЦикла;

КонецПроцедуры


Чтобы просмотреть, какие движения сделал документ, нужно выделить документ в журнале документов и через

меню Действия -> Движения документа.


Если необходимо, чтобы работа пользователя выполнялась в оперативном режиме без лишних вопросов, нужно в

меню Сервис -> Параметры…       на вкладке Оперативный учет.
