Предопределенная процедура происходящая в момент удаления документа.


Процедура ПриУдаленииДокумента(УдаляемыйДокумент, РежимУдаления)

Если НазваниеНабораПрав() = "Администратор" Тогда

Если (УдаляемыйДокумент.Вид() = "ПриходнаяНакладная") И (РежимУдаления = 0) Тогда // помечаем на удаление Приходную накладную и ей соответствующую партию

Если Вопрос("Вы действительно хотите пометить на удаление Приходную накладную", "Да+Нет") = "Да" Тогда

Контрагенты = УдаляемыйДокумент.Контрагенты;

// удаление партию приходной накладной

Партия = СоздатьОбъект("Справочник.Партии");

Спр = СоздатьОбъект("Справочник.Контрагенты");

Спр.НайтиЭлемент(Контрагенты);

Партия.ИспользоватьВладельца(Спр); // в данном примере справочник "Партии" подчинен справочнику "Контрагенты".

Партия.ВыбратьЭлементы(1);

Пока Партия.ПолучитьЭлемент() = 1 Цикл

Если Партия.Документ = УдаляемыйДокумент Тогда // "Документ - это реквизит справочника "Партии".

Партия.Удалить(1); //1 - непосредственное удаление.

КонецЕсли;

КонецЦикла;

КонецЕсли;

КонецЕсли;

КонецЕсли;
