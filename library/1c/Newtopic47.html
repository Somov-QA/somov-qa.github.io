Получить данные из реквизитов шапки внешнего документа


ДокОтпускные = СоздатьОбъект("Документ.НачислениеОтпускных"); //объект агр. типа данных Документ

ДокОтпускные.ВыбратьДокументы(НачМесяца(ДатаДок), КонМесяца(ДатаДок)); //выбираем документ "Начисление Отпускных" из диапазона дат.

Пока ДокОтпускные.ПолучитьДокумент() = 1 Цикл //цикл просмотра всех выбранных документов

Сотрудник = ДокОтпускные.Сотрудник; //получить значение из реквизита. (Сотрудники)

Если НаименованиеФИО = Сотрудник Тогда //если "ФИО" совпадает со значением табличной части документа "Начисление Отпускных" Сотрудник тогда...

Отпускные = ДокОтпускные.НачислОтпускных; //получаем значение начисленые отпускные.

КонецЕсли;

КонецЦикла;
