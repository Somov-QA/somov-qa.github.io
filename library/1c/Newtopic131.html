Методы регистров остатков. Получение остатков.


Остаток(<Измерен1>,<Измерен2>...,<ИмяРесурса>) - Возвращает значение остатка заданного ресурса регистра.

СводныйОстаток(<Измерен1>,<Измерен2>...,<ИмяРесурса>) - Возвращает значение сводного остатка заданного ресурса регистра.

Остатки(<Измерение1>,<Измерение2>...) - Извлечь все ресурсы по регистру.

СводныеОстатки(<Измерение1>,<Измерение2>...) - Извлечь сводные ресурсы по регистру.

ОстаткиПолучить() - Получить все ресурсы по регистру.

ВыбратьДвиженияСОстатками(<ДатаКонца>,<ГрафаОтбора>) - Выбрать все движения регистра с остатками.


Примеры: получение остатков.

Функция ОстатокПоРегистру(ТекТов)

// получить кличество остатка товара на складе.

ОстР = Регистр.ОстаткиТоваров.Остаток(ТекТов, Склад, "Количество"); // ОстаткиТоваров - это идентификатов регистра.

// Остаток(<Измерен1>,<Измерен2>...,<ИмяРесурса>)  // <ИмяРесурса> - название требуемого ресурса регистра.

Возврат ОстР;

КонецФункции

если не указывать "Склад" то получим остатки товара на всех складах.


Пример: Получение количественного и суммового остатко товара на складе

Регистр.ОстаткиТоваров.Остатки(Товар, Склад);

Колво = Регистр.ОстатокТоваров.Количество;

Сумма = Регистр.ОстатокТоваров.Сумма;


Пример: получим значение остатка из "Ресурса" по регистру.

КолВо = Регистр.ОстаткиТоваров.Остаток(Товар, Склад, "Количество");


Пример: получаем значение сводного остатка из "Ресурса" по регистру.

КолВо = Регистр.ОстаткиТоваров.СводныйОстаток(Товар, Склад, "Количество");


Пример: получим значения сводных остатков по регистру.

Регистр.ОстаткиТоваров.СводныеОстатки(Товар, Склад);

КолВо = Регистр.ОстаткиТоваров.Количество;


Пример: получаем все ресурсы регистра по измерениям содержащимся в атрибутах.

Регистр.ОстаткиТоваров.Склад = Склады;

Регистр.ОстаткиТоваров.Товар = Товары;

Регистр.ОстаткиТоваров.ОстаткиПолучить();

КолВо = Регистр.ОстаткиТоваров.Количество;


Пример: произведем выбор всех движений регистра в указанном интервале дат.

Рег = СоздатьОбъект("Регистр.ОстаткиТоваров");

Рег.ВременныйРасчет();

Рег.УстановитьЗначениеФильтра("Склад",Склад);

РассчитатьРегистрыНа(ДатаНач);

Рег.ВыбратьДвиженияСОстатками(ДатаКон);

Пока Рег.ПолучитьДвижение() = 1 Цикл

ТМЦ = Рег.Товар;

КолВо = Рег.Количество;

Сообщить(Строка(ТМЦ) + ":  " + Строка(КолВо));

КонецЦикла;
