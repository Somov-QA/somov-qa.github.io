2. Создание проекта libGDX

libGDX ››

1) Скачать установщик можно с официального сайта libGDX (Download Setup App) http://libgdx.badlogicgames.com/download.html

2) Запустить файл gdx-setup.jar

3) В окне вам необходимо указать название проекта, место расположение где он будет развернут и конечно же путь к Android SDK если вы собираетесь делать мобильное приложение. Рекомендую оставить флаг на версии Desktop которая позволит вам проще работать с отладкой приложения во время разработки.



При нажатии Generate вам могут предложить использовать более новое Android API или же использовать версию предусмотренную libGDX.

В обоих случаях я нажимал NO и использовал предыдущие версии.
