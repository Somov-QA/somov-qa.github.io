3. Импортировать / открыть проект libDGX в Android Studio

libGDX ››

1) Импорт чистого проекта libGDX в Android Studio.

Поле того как чистый проект был сгенерирован запускаем Android Studioи в окне выбора проекта нажимаем Import project (Eclipce ADT, Gradle, etc.)

Открываем папку с чистым проектом и выбираем файл build.gradle нажимаем ОК.



После импорта вам предложат обновить Gradle выполните обновление!


2) Открыть готовый проект.

Запускаем Android Studioи в окне выбора проекта нажимаем Open an existing Android Studio project.

Выбираем папку проекта и нажимаем ОК.



После открытия так же могут предложить обновить Gradle выполните обновление!
