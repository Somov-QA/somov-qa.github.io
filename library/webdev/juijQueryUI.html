Установка jui


Официальное описание: http://www.yiiframework.com/doc-2.0/ext-jui-index.html


Установка jui с помощью Composer


composer.phar require --prefer-dist yiisoft/yii2-jui

Во время установки может потребоваться Token:

  1. Войдите в GitHub
  2. Перейдите в Settings -> Developer settings -> Personal access tokens (прямая ссылка: https://github.com/settings/tokens )
  3. Нажмите кнопку Generate new token (или отредактируйте уже созданный токен)
  4. Флаги при генерации токена можно не устанавливать


Использование jQuery UI в приложении (DatePicker)


<?php


use yii\helpers\Html;

use yii\widgets\ActiveForm;

use yii\jui\DatePicker;


?>


<div class="profile-form">


   <?php $form = ActiveForm::begin(); ?>


   <?= $form->field($model, 'last_activity')->widget(DatePicker::className(), [

           'language' => 'ru-RU',

           'dateFormat' => 'yyyy-MM-dd',

           'clientOptions' => [

               'defaultDate' => '2017-01-01',

               

           ],

           'options' => ['class' => 'form-control']

       ]) ?>


   <div class="form-group">

       <?= Html::submitButton($model->isNewRecord ? 'Create' : 'Update', ['class' => $model->isNewRecord ? 'btn btn-success' : 'btn btn-primary']) ?>

   </div>


   <?php ActiveForm::end(); ?>


</div>
