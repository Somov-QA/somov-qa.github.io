Язык всего приложения

Указать язык приложения можно в конфиге \config\web.php


<?php

$params = require(__DIR__ . '/params.php');

$db = require(__DIR__ . '/db.php');


$config = [

   ...

   'language' => 'ru',

   ...

   ];

}


return $config;


Так же необходимо указать язык в габлоне views\layouts\main.php


<html lang="<?= Yii::$app->language ?>">
