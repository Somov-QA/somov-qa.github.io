Установка Java (JDK, JRE)


1) Установка

Пакет JDK

       sudo apt-get install openjdk-8-jdk


Пакет JRE

       sudo apt-get install openjdk-8-jre


2) Запусл JAR приложений

       java -jar -example.jar
