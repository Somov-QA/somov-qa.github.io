Babel.JS (транспиляция кода)

JavaScript ›› ECMAScript 6 ››

Babel.JS – это транспайлер (транспиляция кода), переписывающий код на ES-2015 в код на предыдущем стандарте ES5.

Ссылка: https://babeljs.io/


Babel для редакторов: https://babeljs.io/docs/editors


Sublime Text 3

Package: https://packagecontrol.io/packages/Babel

GitHub: https://github.com/babel/babel-sublime#installation


1. Установите Package Control: https://packagecontrol.io/installation

2. Нажмите меню Preferences > Package Control

3. Введите команду Package Control: install Package

4. Затем введите Bable и выберите для установки пункт Bable

5. Синтаксис Bable теперь доступен из меню View > Syntax > Bable

Использование Bable


Способ №1: подключить библиотеку Bable и описывать код в теге <script type="text/bable">

<div id="output"></div>
<!-- Load Babel -->
<script src="https://unpkg.com/babel-standalone@6/babel.min.js"></script>
<!-- Your custom script here -->
<script type="text/babel">
       const getMessage = () => "Hello World";
       document.getElementById('output').innerHTML = getMessage();
</script>

так же вы можете подключить внешний скрипт с кодом (но в таком случае должен быть запущен сервер)

<script type="text/babel" src="./js/app.js"></script>


Способ №2: компиляция кода (должен быть установлен NodeJS)


1. Создайте папку project

2. Откройте в папке project консоль

3. Введите команду npm init -y (будет создан файл package.json)

4. Далее необходимо установить Babel https://www.npmjs.com/package/Babel

вводим команду: npm install babel-cli babel-core babel-preset-es2015 --save-dev

5. Создайте в папке project файл index.html

6. Создайте папку src в папке project

7. Создайте в папке src файл scrept.js и запишите строку кода


let message = 'Hello World!';


8. Создайте папку dist в папке project

9. Откройте файл package.json и отредактируйте его в разделе scripts


"scripts": {

   "build": "babel src -d dist --presets es2015"

},


10. В консоли введите команду: npm run build (скрипт будет транспилирован в стандарт ES5 и помещён в папку dist)

11. Чтобы транспиляция выполнялась автоматически при изменении скрипта, необходимо отредактировать файл package.json следующим образом. Нужно добавить watch


"scripts": {

   "build": "babel src -d dist --presets es2015",

   "watch": "babel src -d dist --presets es2015 -w"

},


автоматическая транспиляция запускается командой: npm run watch
