Описание примера

Использование и модификация Bootstrap компонента Input groups - Button addons

Ссылка: https://getbootstrap.com/docs/3.3/components/#input-groups-buttons


В нашем случае мы используем компонент как фильтр по профилю. В компонент будет добавлена кнопка "обновить"


Общий принцип работы компонента:


1. Компонент отображает список профилей загруженный из базы данных или из кэша

2. Кнопка Filter by profile выполняет фильтрацию данных по выранному профилю

3. Кнопка Update выполняет обновление кэша


Общее описание файлов:


search.php - вид для компонента


ProfileSearche.php - основная логика компонента


FormProfileSearch.php - модель компонента


Profile.php - класс созданный с помощью Gii. В этом классе описаны функции  getProfiles() и updateProfileCache()
