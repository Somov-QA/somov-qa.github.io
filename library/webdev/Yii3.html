Установка Yii2


  • Чтобы установить базовый шаблон приложения, выполните команду ниже:

php composer.phar create-project yiisoft/yii2-app-basic basic 2.0.13

composer create-project yiisoft/yii2-app-basic basic 2.0.13

  • Чтобы установить расширенный шаблон приложения, выполните команду ниже:

php composer.phar create-project yiisoft/yii2-app-advanced advanced 2.0.13

composer create-project yiisoft/yii2-app-advanced advanced 2.0.13
