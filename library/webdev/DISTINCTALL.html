DISTINCT/ALL - Повторяющиеся строки

Database ›› SQL ›› Выборка данных ››

Повторяющиеся строки.

DISTINCT – удаляет повторяющиеся строки

ALL – оставляет повторяющиеся строки (по умолчанию)


SELECT DISTINCT elem1, elem2 FROM table1;


SELECT ALL elem1, elem2 FROM table1;
