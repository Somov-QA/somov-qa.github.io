Основные команды


Инициализация

       $ git init


Добавить удалённый репозиторий

       $ git remote add origin https://github.com/CatfishStudio/TestRep.git

       

Получить клон репозитория

       $ git clone https://github.com/CatfishStudio/TestRep.git


Состояния наблюдаемого репозитория

       $ git status

       

Просмотр удалённого репозиторя (узнаем ИМЯ репозитория)

       $ git remote -v

       

Добавить все изменённые файлы в репозитории

       $ git add -A


Добавить определённый файлы

       $ git add README.txt

       

Комментировать добавленные изменённые файлы

       $ git commit -m "...commit..."

       

Пуш. (отправка комментария на сервер)

       $ git push origin master                (origin - ИМЯ репозитория может быть другим)

       

       После чего попросят ввести имя пользователя GIT и пароль

       Username for 'https://github.com':

       Password for 'https://CatfishStudio@github.com':

       

Пул. (получить комментарии с сервера)

       $ git pull origin                                (origin - ИМЯ репозитория может быть другим)


Получить всё с удалённого сервера

       $ git fetch origin                        (origin - ИМЯ репозитория может быть другим)


Команды навигации Linux


pwd                - местонахождение

ls                - содержание каталога

cd d:                - перейти на диск

cd GIT                - перейти в папку

exit                - выход из консоли
