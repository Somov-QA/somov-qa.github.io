Пространства имён

ActionScript3 ››

Пространства имен позволяют управлять доступностью создаваемых свойств и методов. Можно рассматривать спецификаторы управления доступом public, private, protected и internal как встроенные пространства имен.

При использовании пространств имен необходимо выполнить три основных шага.

1. Прежде всего необходимо определить пространство имен, используя ключевое слово namespace. Например, определяется пространство имен version1.

       namespace version1;


2. Во-вторых, необходимо применить пространство имен, используя его вместо спецификатора контроля доступа при объявлении свойства или метода.

В следующем примере функция myFunction() размещается в пространстве имен version1.

       version1 function myFunction():void {}


3. В-третьих, как только пространство имен будет применено, к нему можно адресоваться с помощью директивы use или уточняя имя идентификатора указанием пространства имен.

В следующем примере myFunction() вызывается с использованием директивы use.

       use namespace version1;

       myFunction();

Для вызова функции myFunction() можно также использовать уточненное имя, как показано в следующем примере (но так лучше не делать)

       version1::myFunction();



Определение пространств имен

       namespace flash_proxy = "<http://www.adobe.com/flash/proxy>";

       package flash.utils

       {

               public namespace flash_proxy;

       }


Применение пространств имен

       namespace example1;

       class someClass

       {

               example1 myFunction() {}

       }


Привязка пространств имен

       use namespace example1;

       myFunction();


Использование пространств имен

       package flash.utils

       {

               public namespace flash_proxy;

       }

       public class Proxy

       {

               flash_proxy function callProperty(name:*, ... rest):*

               flash_proxy function deleteProperty(name:*):Boolean

               ...

       }


       package

       {

               import flash.utils.Proxy;

               import flash.utils.flash_proxy;

               dynamic class MyProxy extends Proxy

               {

                       flash_proxy override function callProperty(name:*, ...rest):*

                       {

                               trace("method call intercepted: " + name);

                       }

               }

       }
