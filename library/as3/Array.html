Вставка элементов массива

Метод push() добавляет один или несколько элементов в конец массива.

Метод unshift() вставляет один или более элементов в начало массива/

Метод splice() вставляет любое число элементов по заданному индексу в массиве.

       var planets:/*String*/Array = [];

       planets.push("Mars");

       planets.unshift("Mercury");

       planets.splice(1, 0, "Venus", "Earth");

       trace(planets);        // результат: Mercury,Venus,Earth,Mars


Удаление элементов массива

Метод pop() удаляет элемент из конца массива.

Метод shift() удаляет элемент из начала массива.

Метод splice(), который также может использоваться для вставки элементов, удаляет произвольное число элементов, начиная с индекса, заданного первым из передаваемых методу аргументов.

       var oceans:/*String*/Array = ["Victoria", "Pacific", "Aral", "Superior", "Indian",

                                                                               "Huron"];

       oceans.splice(2, 2, "Arctic", "Atlantic");

       oceans.pop();

       oceans.shift();

       trace(oceans); // результат: Pacific,Arctic,Atlantic,Indian


Сортировка массива

reverse(), sort() и sortOn(), - позволяющих изменить порядок элементов в индексном массиве путем сортировки или задания обратного порядка.

reverse() Изменяет порядок элементов таким образом, что последний элемент становится первым, предпоследний - вторым и так далее.

sort() Позволяет сортировать элементы объекта Array разными предварительно определенными способами, например в алфавитном порядке или по номерам.

sortOn() Позволяет сортировать объекты, которые имеют одно или несколько общих свойств.


Запросы в массив

concat(), join(), slice(), toString() - отправляют запросы в массив, но не изменяют его.

Методы concat() и slice() возвращают новые массивы

Методы join() и toString() возвращают строки

public static function CreateCellArrayMatrix2D(columns:int, rows:int):Array /*Cell*/

{

       /* i - столбец; j - строка */

       var newArray:/*Cell*/Array = [];

       var unitAdd:Cell = new Cell();

       for (var i:int = 0; i < columns; i++)

       {

               var newRow:Array = [];

               for (var j:int = 0; j < rows; j++)

               {

                       newRow.push(unitAdd);

               }

               newArray.push(newRow);

       }

       return newArray;

}
