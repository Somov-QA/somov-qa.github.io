int[] arrInt;

- Многомерные массивы

- Класса Array (System.Array)

CreateInstance() - позволяет создавать многомерный или одномерный массив.

int[] lengths = {10};

int lowBounds = {1};

Array aMatrix = Array.CreateInstance(typeof(int), lengths, lowBounds);


- Класс ArrayList (System.Collections)

Предназначен для хранения объекточ произвольного типа

static ArrayList Books = new ArrayList();

Books.Add("Значение");

string nameBook = Books[0].toString();


  Add - добавить новый объект в конец списка

  AddRange - добавить несколько объектов в конец списка

  BinarySearch - поиск объекта в отсортированном списке

  Capacity - текущая ёмкость коллекции

  Clean - удалить все элементы

  Contains - возвращает истину если коллекция содержит элемент

  Count - текущая длина коллекции

  GetRange - возвращает диапазон элеменов

  IndexOf - возвращает индекс элемента со значением

  Insert - вставляет элемент по интексу в коллекцию

  LastIndexOf - индекс последнего вхождения

  Remove - удаляет из коллекции первое вхождение

  RemoveAt - удаляет элемент по индексу

  RemoveRange - удаляет диапазон элементов

  Reverse - реверсирует элементы

  Sort - сортирует элементы
