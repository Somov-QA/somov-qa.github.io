Установка NodeJS

Ubuntu ››

1) Установка

       sudo apt-get update

       sudo apt-get install nodejs


2) Переключиться на работу с сервером

       nodejs


3) Команды для проверки работы сервера

       console.log('hello');

или

       2+3


4) Для того, чтобы выйти из интерактивной оболочки,

необходимо просто нажать Ctrl + C (нажать дважды).
