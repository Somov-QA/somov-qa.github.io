Установка VSCode

Ubuntu ››

1) Для запуска необходимо установить пакет JDK

       sudo apt-get install openjdk-8-jdk

----------------------------------------------------------------------------------------------------------

2) Необходимо установить NodeJS

       sudo apt-get update

       sudo apt-get install nodejs

----------------------------------------------------------------------------------------------------------

3) На сайте https://code.visualstudio.com

скачиваем deb файл установки vscode


4) Установка gdebi для DEB пакета

       sudo apt-get install gdebi


5) Установка DEB пакета

       sudo dpkg -i code_1.7.2-1479767514_i386.deb

----------------------------------------------------------------------------------------------------------

6) Установка пакетов Node

       npm install

----------------------------------------------------------------------------------------------------------

7) Запуск задачи в vscode

       ctrl+p

вводим команду

       task dev

Grunt не установлен в вашей системе. Чтобы установить его, выполните команду

       npm install -g grunt.

----------------------------------------------------------------------------------------------------------

8) Кнопка F1 и команда theme
