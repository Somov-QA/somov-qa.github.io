Рекордер

ПО автоматизированного тестирования ›› Android SDK ›› Тестирование с помощью MonkeyRunner ››

Для записи и воспроизведения сделанных тестером действий используются специально для этого созданные средства: monkey_recorder.py и monkey_playback.py

Эти скрепты можно скачать по адресу: https://github.com/miracle2k/android-platform_sdk/tree/master/monkeyrunner/scripts


Копируем папку scripts на дист C:


Запуск рекордера с помощью следующей команды:

monkeyrunner c:\logs\scripts\monkey_recorder.py

произведённые действия сохраняем в файл: script.out


Воспроизведение записанных действий

       monkeyrunner c:\logs\scripts\monkey_playback.py c:\logs\script.out
