Чтобы воспользоваться эмулятором сначала нужно настроить AVD

Для этого запустим AVD Manager который находится по следующему адресу:

C:\Program Files (x86)\Android\android-sdk



Настроив виртуальное устройство его можно запустить при помощи кнопки Start

Установка apk на эмулятор https://youtu.be/Xgx01zJujzE

c:\builds>adb install file.appk
