Тестирование с помощью getevent/sendevent

ПО автоматизированного тестирования ›› Android SDK ››

Чтобы записать события, генерируемые пользователем, надо выполнить следующую команду:

       adb shell getevent -t > C:/TESTING/events/getevent.txt


в результате будет получена следующая информация

/dev/input/event0 - это тип устройства которое генерирует событие

0003 0039 00000127 - это параметры событий в 16-ти ричном виде. Для повтора этих параметров 16-ти ричное значение нужно конвертировать в десятичную систему счисления.


Конвертор: http://www.translatorscafe.com/cafe/RU/units-converter/numbers/4-3/


После конвертации можно повторить команды таким образом:

       adb shell sendevent /dev/input/event0 3 57 295

       adb shell sendevent /dev/input/event0 3 58 21

       adb shell sendevent /dev/input/event0 3 53 224

       adb shell sendevent /dev/input/event0 3 54 145

       adb shell sendevent /dev/input/event0 0 0 0
