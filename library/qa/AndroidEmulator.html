Настройка Android Emulator

Описание на официальном сайте: https://developer.android.com/studio/run/managing-avds.html


1. Запустите Android Studio и создайте проект


2. Откройте AVD Manager нажав кнопку на панели инструментов или через меню Tools > Android > AVD Manager



Откроется окно виртуальных девайсов



Нажмите кнопку Create Virtual Device чтобы создать новый девайс

Выберите тип устройства. Нажмите кнопку Next.



Выберите операционную систему устройства.

Если нужной системы нет нажмите на Download чтобы докачать необходимую систему.

Нажмите кнопку Next.



Укажите имя устройства, оно станит идентификатором. Нажмите кнопку Finish.



Устройство создано.
