Запуск Android Emulator

Способ №1: Запустить из Android Studio


1. Запустите Android Studio и создайте проект


2. Откройте AVD Manager нажав кнопку на панели инструментов или через меню Tools > Android > AVD Manager



Откроется окно виртуальных девайсов



Нажмите на кнопку launch напротив нужного устройства


Эмулятор устройства будет запущен


Способ №2: Запуск эмулятора из командной строки


Откройте консоль и перейдите в папку C:\Users\Cartish\AppData\Local\Android\tools\


Чтобы увидеть список доступных устройств введите команду


emulator -list-avds


Чтобы запустить нужное устройство введите команду:


emulator -avd Nexus_7_2012_Android_5




Запустится эмулятор устройства
