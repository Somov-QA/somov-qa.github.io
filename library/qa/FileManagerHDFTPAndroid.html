File Manager HD(FTP) (Android)

ПО для тестирования ››

Полнофункциональный инструмент управления файлами! 
Разработан командой Clean Master

Основные возможности File Manager
☆ Полный функционал: вырезать, копировать, вставить, удалить, сжать, распаковать, поиск и многое другое.
☆ Поддержка облака: переключение между локальными и облачными файлами - поддержка Dropbox, Box, диска Google, SkyDrive и многих других.
☆ Передача по Wi-Fi: используйте Ваш компьютер для работы с файлами с помощью FTP.
☆ Удобство: просмотр файлов по типу (изображения, аудио, видео, последние, загрузки и т.д.), использование виджета для доступа к папкам в одно касание на домашнем экране.
☆ Мультипротокол: передача файлов с помощью FTP, FTPS, SFTP, WebDAV, LAN/SMB, потоковая передача медиа с LAN/SMB/FTP/WebDAV/облака без загрузки.
