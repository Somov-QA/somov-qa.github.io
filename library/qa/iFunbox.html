iFunbox

ПО для тестирования ›› iOS ››

Install iOS Apps from customized or unsigned IPA package.

iFunbox allows you install IPA packages from in-house distribution, which is produced with Apple's Developer Enterprise Program.

It also allows you install *.ipa files archieved from an App under development or cracked by you.
