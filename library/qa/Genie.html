АВТОМАТИЗАЦИЯ ТЕСТИРОВАНИЯ: GENIE

Automated UI Tester for Adobe® ActionScript® is a pure ActionScript 3.0 SWF automation testing tool with the capability of recording user actions on an SWF file and playing them back with high fidelity. The API for this tool is provided with name “Genie.”


Genie: Необходимое ПО

1. Скачать Genie (архив Genie-binary.zip)

Официальный сайт: http://sourceforge.net/adobe/genie/wiki/Home/


2. Установить дебажный Flash плеер.

Официальный сайт: http://www.adobe.com/support/flashplayer/downloads.html


3. Установить JDK - Java Development Kit

Официальный сайт: http://www.oracle.com/technetwork/java/javase/downloads/index.html


4. Скачать среду разработки Eclipse

Официальный сайт: http://www.eclipse.org/downloads/


5. Настройка Global Security Settings panel для Flash преера

Официальный сайт Flash Player Help: http://www.macromedia.com/support/documentation/en/flashplayer/help/settings_manager04.html
