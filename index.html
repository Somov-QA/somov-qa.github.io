Разработки


FastADB
Программа позволяет работать с Android Debug Bridge
более удобно и быстрее: снимать логи и скриншоты,
устанавливать и удалять приложения и пр.

DetectGaYm
Программа для автоматизации тестирования
событий Google analytics и Yandex metrika.
Проверка событияв в момент отправки данных из web форм.

* Более подробная информация о разработках смотрите в разделе подробнее >>


TestSitemap
Программа предназначена для проверки ссылок в файле sitemap.
Программа циклично проходит по всем ссылкам
и проверяет 200-й статус страниц.

SearchSiteContent
Программа для выполнения поиска заданного контента
на страницах сайта указанных в файле sitemap.
В результате выводится отчет страниц с найденным контентом.

JsonFileEditor
Программа предназначена для редактирования json файлов.
Удобный интерфейс позволяет комфортно работать с данными.
