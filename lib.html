Справочники

C#

Объектно-ориентированный язык программирования. Язык для разработки приложений на платформе Microsoft .NET Framework.

PHP

Препроцессор гипертекста, инструменты для создания персональных веб-страниц, скриптовый язык.

JavaScript

Прототипно-ориентированный сценарный язык программирования. Является реализацией языка ECMAScript.

TypeScript

Язык программирования, представленный Microsoft в 2012 году и позиционируемый как средство разработки веб-приложений, расширяющее возможности JavaScript.

ActionScript3

Объектно-ориентированный язык программирования, один из диалектов ECMAScript, который добавляет интерактивность, обработку данных и многое другое в содержимое Flash-приложений.

SQL

Формальный непроцедурный язык программирования, применяемый для создания, модификации и управления данными в произвольной реляционной базе данных, управляемой соответствующей системой управления базами данных.

Phaser

Фреймворк для создания игр под html5.

PixiJS

Фреймворк для создания игр под html5.

Defold

Мультиплатформенный 2D игровой движок от компании King.

libGDX

Фреймворк для создания игр и приложений, написанный на Java с использованием C и C++ (для более быстрой работы). Он позволяет писать кроссплатформенные игры и приложения используя один код.

AJAX

Асинхронный JavaScript и XML. Это эффективный способ совместного использования HTML, CSS, JavaScript и DOM.

Git

Справочник команд для работы с Git.

Linux

Справочник команд для работы в консоли Linux.

MsDos

Справочник команд для работы в консоли MsDos.
