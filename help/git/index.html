GIT команды

Инструкция:

1. Добавить репозиторий (Add or Create...)
$ git clone https://...git
$ git remote add origin https://...git

2. Состояния наблюдаемого репозитория
$ git status

3. Список всех веток (определить текущую ветку)
git branch

4. Добавление ветки (Add Branch...)
$ git branch MyNewBranch

5. Переключение между веток
$ git checkout MyNewBranch
$ git checkout master

6. Слитие ветки (Merge...)
Сначала нужно переключиться в ветку master,
а потом слить вертку MyNewBranch, в этом случае ветка MyNewBranch будет слита в ветку master
$ git merge MyNewBranch

7. Удаление ветки (Delete...)
$ git branch -d MyNewBranch

8. Отменить изменение файла (Discard...)
$ git checkout -- file.txt

9. Отметить все изменения
$ git reset --hard

10. Добавить изменения
$ git add -A

11. Закомитить правки (Commit...)
$ git commit -m “текст комментария”

12. Отправить комит (Push)
$ git push origin MyNewBranch
$ git push origin master

13. Получить комит (Pull)
$ git pull origin
