📘 Playwright: класс Page

Полный справочник всех методов, свойств, событий и устаревших API из официальной документации.
Основано на <div class="col col--3"> — актуально для Playwright 1.46+

📌 Методы (Methods) — 74 метода

▼
addInitScript
Добавляет скрипт, выполняемый перед загрузкой страницы / фреймов. Полезен для моков.
await page.addInitScript(() => { window.mockData = { id: 42 } });
addLocatorHandler
Регистрирует обработчик для неожиданных оверлеев (например, диалогов). Вызывается перед любым действием.
await page.addLocatorHandler(page.getByText('Cookie banner'), async () => { await page.getByRole('button', { name: 'Accept' }).click(); });
addScriptTag
Внедряет тег <script> на страницу (по URL, пути или raw-коду).
await page.addScriptTag({ url: 'https://code.jquery.com/jquery-3.6.0.min.js' });
addStyleTag
Добавляет CSS-стили через тег <style> или внешний файл.
await page.addStyleTag({ content: 'body { background: #f0f0f0; }' });
ariaSnapshot
Возвращает текстовое представление дерева доступности (ARIA snapshot).
const snapshot = await page.locator('body').ariaSnapshot(); console.log(snapshot);
bringToFront
Переводит текущую страницу (вкладку) на передний план.
await page.bringToFront();
cancelPickLocator
Отменяет режим инспектора locator (page.pickLocator).
await page.cancelPickLocator();
clearConsoleMessages
Очищает список перехваченных консольных сообщений (page.consoleMessages).
await page.clearConsoleMessages();
clearPageErrors
Очищает список ошибок страницы (page.pageErrors).
await page.clearPageErrors();
close
Закрывает страницу (вкладку). Генерирует событие 'close'.
await page.close();
consoleMessages
Возвращает массив всех консольных сообщений, записанных на странице.
const messages = page.consoleMessages(); messages.forEach(msg => console.log(msg.text()));
content
Возвращает HTML-содержимое документа (полный outerHTML).
const html = await page.content(); console.log(html.substring(0, 100));
context
Возвращает браузерный контекст (BrowserContext), которому принадлежит страница.
const context = page.context();
dragAndDrop
Выполняет drag-and-drop из одного элемента в другой.
await page.dragAndDrop('#source', '#target');
emulateMedia
Эмулирует медиа-характеристики: цветовая схема, предпочтение уменьшенного движения и т.д.
await page.emulateMedia({ colorScheme: 'dark' });
evaluate
Выполняет JavaScript-функцию внутри страницы, возвращает результат.
const title = await page.evaluate(() => document.title);
evaluateHandle
Выполняет JS и возвращает JSHandle на объект (для дальнейшей работы).
const windowHandle = await page.evaluateHandle(() => window);
exposeBinding
Экспонирует привязку (binding) на страницу — функцию, вызываемую из браузера.
await page.exposeBinding('getVersion', () => '1.0.0');
exposeFunction
Экспонирует функцию в window для вызова из браузера с возвратом Promise.
await page.exposeFunction('add', (a, b) => a + b);
frame
Возвращает фрейм по имени или URL-селектору.
const frame = page.frame('iframe-name'); await frame.fill('input', 'text');
frameLocator
Создаёт Locator для элементов внутри указанного фрейма (iframe).
await page.frameLocator('#myIframe').getByText('Submit').click();
frames
Возвращает массив всех фреймов на странице (главный + дочерние).
const frames = page.frames(); console.log(`Total frames: ${frames.length}`);
getByAltText
Локатор по атрибуту alt (для изображений, областей).
await page.getByAltText('Playwright logo').click();
getByLabel
Локатор по тексту метки (label) элемента.
await page.getByLabel('User Name').fill('John');
getByPlaceholder
Локатор по плейсхолдеру input/textarea.
await page.getByPlaceholder('Search...').fill('Playwright');
getByRole
Локатор по роли ARIA (button, textbox, checkbox …).
await page.getByRole('button', { name: 'Submit' }).click();
getByTestId
Локатор по атрибуту data-testid (или кастомному).
await page.getByTestId('submit-button').click();
getByText
Локатор по текстовому содержимому.
await page.getByText('Welcome, John').click();
getByTitle
Локатор по атрибуту title.
await page.getByTitle('Close').click();
goBack
Переходит на предыдущую страницу в истории (аналог кнопки "Назад").
await page.goBack();
goForward
Переходит на следующую страницу в истории.
await page.goForward();
goto
Переходит по URL, дожидается загрузки (событие load).
await page.goto('https://example.com');
hideHighlight
Скрывает подсветку элементов, включенную через highlight().
await page.hideHighlight();
isClosed
Проверяет, закрыта ли уже страница.
if (page.isClosed()) { console.log('Page is closed'); }
locator
Создаёт локатор для поиска элементов (основной современный API).
await page.locator('button.submit').click();
mainFrame
Возвращает главный фрейм страницы.
const main = page.mainFrame(); await main.fill('input', 'text');
opener
Возвращает страницу, которая открыла текущую (через window.open).
const opener = await page.opener(); if (opener) await opener.bringToFront();
pageErrors
Возвращает массив необработанных ошибок JavaScript на странице.
const errors = page.pageErrors(); console.log(errors.map(e => e.message));
pause
Приостанавливает выполнение теста и открывает инспектор Playwright.
await page.pause();
pdf
Генерирует PDF-файл страницы (работает только в Chromium).
await page.pdf({ path: 'page.pdf', format: 'A4' });
pickLocator
Запускает интерактивный выбор локатора на странице (режим инспектора).
await page.pickLocator();
reload
Перезагружает текущую страницу.
await page.reload();
removeAllListeners
Удаляет всех слушателей событий страницы (унаследовано от EventEmitter).
page.removeAllListeners();
removeLocatorHandler
Удаляет ранее зарегистрированный обработчик локатора.
await page.removeLocatorHandler(handlerId);
requestGC
Принудительно запускает сборку мусора на странице (экспериментально).
await page.requestGC();
requests
Возвращает массив всех сетевых запросов, выполненных на странице.
const requests = page.requests(); console.log(requests.map(r => r.url()));
route
Перехватывает и модифицирует сетевые запросы (маршрутизация).
await page.route('**/api/**', route => route.fulfill({ status: 200, body: 'mocked' }));
routeFromHAR
Перехватывает запросы с использованием HAR-архива (mock-данные).
await page.routeFromHAR('recording.har');
routeWebSocket
Перехватывает WebSocket-соединения и сообщения.
await page.routeWebSocket('wss://example.com/ws', ws => ws.onMessage(msg => console.log(msg)));
screenshot
Делает скриншот страницы (область, полная страница, маскировка).
await page.screenshot({ path: 'screenshot.png', fullPage: true });
setContent
Заменяет всё содержимое страницы переданным HTML-кодом.
await page.setContent('<h1>Hello</h1>');
setDefaultNavigationTimeout
Устанавливает таймаут для всех навигаций (goto, reload, goBack).
page.setDefaultNavigationTimeout(60000);
setDefaultTimeout
Устанавливает глобальный таймаут для всех асинхронных операций на странице.
page.setDefaultTimeout(30000);
setExtraHTTPHeaders
Добавляет дополнительные HTTP-заголовки ко всем запросам страницы.
await page.setExtraHTTPHeaders({ 'X-Custom': 'value' });
setViewportSize
Изменяет размер вьюпорта (ширина/высота) страницы.
await page.setViewportSize({ width: 1280, height: 720 });
title
Возвращает заголовок страницы (document.title).
const title = await page.title(); console.log(title);
unroute
Отменяет перехват запросов, созданный через page.route.
await page.unroute('**/api/**');
unrouteAll
Удаляет все маршруты, зарегистрированные на странице.
await page.unrouteAll();
url
Возвращает текущий URL страницы.
const currentUrl = page.url(); console.log(currentUrl);
video
Возвращает объект Video для записи экрана (если контекст создан с опцией recordVideo).
const video = page.video(); await video.saveAs('recording.webm');
viewportSize
Возвращает текущий размер вьюпорта (объект width/height).
const size = page.viewportSize(); console.log(`${size.width}x${size.height}`);
waitForEvent
Ожидает определённое событие на странице (например, 'request').
await page.waitForEvent('request', request => request.url().includes('/api'));
waitForFunction
Ожидает, пока переданная функция на странице не вернёт truthy-значение.
await page.waitForFunction(() => document.querySelector('.loaded'));
waitForLoadState
Ожидает определённого состояния загрузки ('load', 'domcontentloaded', 'networkidle').
await page.waitForLoadState('networkidle');
waitForRequest
Ожидает выполнения сетевого запроса по URL или предикату.
await page.waitForRequest('**/api/data');
waitForResponse
Ожидает получения ответа на запрос (по url/предикату).
await page.waitForResponse('**/api/data');
waitForURL
Ожидает, пока URL страницы не совпадёт с заданным (частично или полностью).
await page.waitForURL('**/dashboard');
workers
Возвращает массив веб-воркеров (Web Workers), созданных на странице.
const workers = page.workers(); console.log(`Active workers: ${workers.length}`);

⚙️ Свойства (Properties)

▼
clock
Управление временем (эмуляция часов, ускорение времени) для тестов, зависящих от времени.
await page.clock.install({ time: new Date('2024-01-01') });
coverage
Сбор информации о покрытии кода JS/CSS (только Chromium).
await page.coverage.startJSCoverage(); // ... await page.coverage.stopJSCoverage();
keyboard
Объект для эмуляции клавиатурного ввода (нажатия, отпускания, ввод текста).
await page.keyboard.press('Enter');
mouse
Эмуляция мыши: перемещение, клики, скролл, нажатие кнопок.
await page.mouse.click(100, 200);
request
Объект APIRequest для выполнения HTTP-запросов из контекста страницы (cookies наследуются).
const response = await page.request.get('https://api.example.com/data');
screencast
Управление записью экрана страницы (экспериментальное).
await page.screencast.start({ path: 'recording.webm' });
touchscreen
Эмуляция сенсорных событий (tap, swipe).
await page.touchscreen.tap(100, 200);

🎯 События (Events)

▼
on('close')
Срабатывает, когда страница закрывается (вкладка уничтожена).
page.on('close', () => console.log('Page closed'));
on('console')
Возникает при любом вызове console.log(), console.error() и т.д.
page.on('console', msg => console.log(`Console: ${msg.text()}`));
on('crash')
Страница или её процесс аварийно завершились.
page.on('crash', () => console.log('Page crashed!'));
on('dialog')
При появлении нативного диалога (alert, confirm, prompt).
page.on('dialog', dialog => dialog.accept('OK'));
on('domcontentloaded')
Когда DOM готов, не дожидаясь стилей/картинок.
page.on('domcontentloaded', () => console.log('DOM ready'));
on('download')
Начинается загрузка файла (скачивание).
page.on('download', download => download.saveAs('./file.pdf'));
on('filechooser')
Открыт системный диалог выбора файла (input type="file").
page.on('filechooser', fileChooser => fileChooser.setFiles('file.txt'));
on('frameattached')
Новый фрейм (iframe) присоединён к странице.
page.on('frameattached', frame => console.log(`Frame attached: ${frame.url()}`));
on('framedetached')
Фрейм удалён со страницы.
page.on('framedetached', frame => console.log(`Frame detached: ${frame.url()}`));
on('framenavigated')
Фрейм перешёл по новому URL.
page.on('framenavigated', frame => console.log(`Frame navigated: ${frame.url()}`));
on('load')
Страница полностью загружена (все ресурсы).
page.on('load', () => console.log('Page loaded'));
on('pageerror')
Необработанное исключение JavaScript (unhandled exception).
page.on('pageerror', error => console.log(`Error: ${error.message}`));
on('popup')
Открыто новое окно/вкладка (window.open).
page.on('popup', popup => console.log(`Popup opened: ${popup.url()}`));
on('request')
Отправлен HTTP-запрос (событие до получения ответа).
page.on('request', request => console.log(`Request: ${request.url()}`));
on('requestfailed')
Запрос завершился ошибкой (нет сети, таймаут, 4xx/5xx? нет — техническая ошибка).
page.on('requestfailed', request => console.log(`Request failed: ${request.url()}`));
on('requestfinished')
Запрос успешно завершён (ответ получен).
page.on('requestfinished', request => console.log(`Request finished: ${request.url()}`));
on('response')
Получен HTTP-ответ (включая статус, заголовки).
page.on('response', response => console.log(`Response: ${response.status()} ${response.url()}`));
on('websocket')
Создано новое WebSocket-соединение.
page.on('websocket', ws => console.log('WebSocket created'));
on('worker')
Создан новый веб-воркер.
page.on('worker', worker => console.log(`Worker created: ${worker.url()}`));

⚠️ Устаревшие методы (Deprecated) заменены на Locator API

▼
$ deprecated
Поиск одного элемента по селектору (возвращает ElementHandle). Используйте locator().
// Вместо const el = await page.$('#id'); // Используйте: const el = page.locator('#id');
$$ deprecated
Поиск всех элементов по селектору → массив ElementHandle[].
// Вместо const els = await page.$$('.item'); // Используйте: const els = page.locator('.item').all();
$eval deprecated
Выполняет JS на первом элементе, найденном по селектору.
// Вместо const value = await page.$eval('#input', el => el.value); // Используйте: const value = await page.locator('#input').inputValue();
$$eval deprecated
Выполняет JS на всех элементах, найденных по селектору.
// Вместо const texts = await page.$$eval('.item', items => items.map(i => i.textContent)); // Используйте: const texts = await page.locator('.item').allTextContents();
check deprecated
Ставит галочку в чекбоксе. Заменено на locator().check().
// Вместо await page.check('#agree'); // Используйте: await page.locator('#agree').check();
click deprecated
Клик по элементу. Используйте locator().click().
// Вместо await page.click('#submit'); // Используйте: await page.locator('#submit').click();
dblclick deprecated
Двойной клик. Заменён на locator().dblclick().
// Вместо await page.dblclick('#element'); // Используйте: await page.locator('#element').dblclick();
dispatchEvent deprecated
Искусственно генерирует DOM-событие.
// Вместо await page.dispatchEvent('#btn', 'click'); // Используйте: await page.locator('#btn').dispatchEvent('click');
fill deprecated
Заполняет поле ввода текстом. Используйте locator().fill().
// Вместо await page.fill('#name', 'John'); // Используйте: await page.locator('#name').fill('John');
focus deprecated
Устанавливает фокус на элемент. Заменено на locator().focus().
// Вместо await page.focus('#input'); // Используйте: await page.locator('#input').focus();
getAttribute deprecated
Возвращает значение атрибута элемента. Используйте locator().getAttribute().
// Вместо const attr = await page.getAttribute('#link', 'href'); // Используйте: const attr = await page.locator('#link').getAttribute('href');
hover deprecated
Наводит курсор на элемент. Заменено на locator().hover().
// Вместо await page.hover('#menu'); // Используйте: await page.locator('#menu').hover();
innerHTML deprecated
Получает innerHTML элемента. Используйте locator().innerHTML().
// Вместо const html = await page.innerHTML('#div'); // Используйте: const html = await page.locator('#div').innerHTML();
innerText deprecated
Текстовое содержимое элемента (innerText). Заменён на locator().innerText().
// Вместо const text = await page.innerText('#msg'); // Используйте: const text = await page.locator('#msg').innerText();
inputValue deprecated
Значение поля ввода. Используйте locator().inputValue().
// Вместо const value = await page.inputValue('#email'); // Используйте: const value = await page.locator('#email').inputValue();
isChecked deprecated
Проверяет, отмечен ли чекбокс/радио. Заменено на locator().isChecked().
// Вместо const checked = await page.isChecked('#agree'); // Используйте: const checked = await page.locator('#agree').isChecked();
isDisabled deprecated
Проверяет, отключён ли элемент. Используйте locator().isDisabled().
// Вместо const disabled = await page.isDisabled('#btn'); // Используйте: const disabled = await page.locator('#btn').isDisabled();
isEditable deprecated
Доступен ли для редактирования. Заменён на locator().isEditable().
// Вместо const editable = await page.isEditable('#field'); // Используйте: const editable = await page.locator('#field').isEditable();
isEnabled deprecated
Включён ли элемент (не disabled). Аналог locator().isEnabled().
// Вместо const enabled = await page.isEnabled('#btn'); // Используйте: const enabled = await page.locator('#btn').isEnabled();
isHidden deprecated
Скрыт ли элемент. Используйте locator().isHidden().
// Вместо const hidden = await page.isHidden('#modal'); // Используйте: const hidden = await page.locator('#modal').isHidden();
isVisible deprecated
Видим ли элемент. Заменено на locator().isVisible().
// Вместо const visible = await page.isVisible('#modal'); // Используйте: const visible = await page.locator('#modal').isVisible();
press deprecated
Нажимает клавишу/сочетание клавиш. Используйте locator().press().
// Вместо await page.press('#input', 'Enter'); // Используйте: await page.locator('#input').press('Enter');
selectOption deprecated
Выбирает option в select. Заменено на locator().selectOption().
// Вместо await page.selectOption('#select', 'value'); // Используйте: await page.locator('#select').selectOption('value');
setChecked deprecated
Устанавливает состояние чекбокса/радио. Используйте locator().setChecked().
// Вместо await page.setChecked('#agree', true); // Используйте: await page.locator('#agree').setChecked(true);
setInputFiles deprecated
Загружает файлы в input[type=file]. Заменён на locator().setInputFiles().
// Вместо await page.setInputFiles('#upload', 'file.txt'); // Используйте: await page.locator('#upload').setInputFiles('file.txt');
tap deprecated
Касание (tap) на сенсорном экране. Используйте locator().tap().
// Вместо await page.tap('#element'); // Используйте: await page.locator('#element').tap();
textContent deprecated
Возвращает textContent элемента. Аналог locator().textContent().
// Вместо const text = await page.textContent('#title'); // Используйте: const text = await page.locator('#title').textContent();
type deprecated
Печатает текст с задержками. Заменён на locator().pressSequentially().
// Вместо await page.type('#input', 'hello'); // Используйте: await page.locator('#input').pressSequentially('hello');
uncheck deprecated
Снимает галочку с чекбокса. Используйте locator().uncheck().
// Вместо await page.uncheck('#agree'); // Используйте: await page.locator('#agree').uncheck();
waitForNavigation deprecated
Ожидает навигации. Рекомендуется использовать page.waitForURL() или expect(page).toHaveURL().
// Вместо await page.waitForNavigation(); // Используйте: await page.waitForURL('**/target');
waitForSelector deprecated
Ожидает появления/исчезновения селектора. Заменён на locator().waitFor().
// Вместо await page.waitForSelector('#element'); // Используйте: await page.locator('#element').waitFor();
waitForTimeout deprecated
Приостанавливает выполнение на заданное время (не рекомендуется). Используйте вместо этого явные ожидания (waitForSelector, waitForLoadState).
// Не рекомендуется: await page.waitForTimeout(1000); // Вместо этого: await page.waitForLoadState('load');
