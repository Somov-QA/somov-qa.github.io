6. Смена экрана

Практика libGDX ››

Чтобы добавить сцену необходимо создать класс описанной ранее сцена и добавить её в классе Game с помощью метода setScreen()

package com.mygdx.game;


import com.badlogic.gdx.Game;

import com.mygdx.game.menu.Menu;


public class MyGdxGame extends Game {


  public Menu menu;


@Override

public void create () {

     menu = new Menu(this);

     setScreen(menu);

}


...

Чтобы сменить экран нужно иметь ссылку на основной класс Game и так же с помощью метода setScreen() заменить экран на новый.


button.addListener(new ChangeListener() {

   @Override

public void changed (ChangeListener.ChangeEvent event, Actor actor) {

       game.setScreen(new TestObj(game));

       dispose();

}

});


В результате отобразится новый экран, а прежний будет уничтожен.
