Масштабирование Desktop / Mobile

Включить масштабирование. (Применяется в Boot.ts)

if (this.game.device.desktop) {

      // это исправит ошибку в IE Edge

      this.stage.disableVisibilityChange = true;

      // автоматически масштабирует игру, но сохраняет исходное соотношение сторон

      this.scale.scaleMode = Phaser.ScaleManager.SHOW_ALL;

      this.scale.pageAlignHorizontally = true;

      this.game.scale.windowConstraints.bottom = 'visual';

} else {

      // пользовательский масштаб для мобильных устройств, чтобы сохранить пользовательские окна просмотра

      this.scale.scaleMode = Phaser.ScaleManager.USER_SCALE;

      // изменить размер

      window.addEventListener('resize', (e: Event) => this.mobileResizeCallback(this.game.scale));

      this.game.scale.onSizeChange.add(() => {

             this.game.state.getCurrentState().resize();

      }, this);

      this.mobileResizeCallback(this.game.scale);

}





Размеры и масштабирование игры

Стартовые размеры игры 1980х1024 , определяются в классе Constants (Data\Constants.ts)

Размеры игры масштабируются, под разрешение устройства юзера.

Параметры масштабирования изначально задаются в SunsetSolitaire.Boot.init() Отдельно для десктопа и для мобильных устройств.

Для десктопа используется стандартный scale mode Phaser http://phaser.io/docs/2.4.7/Phaser.ScaleManager.html#.SHOW_ALL

Для мобильных устройств - кастомный http://phaser.io/docs/2.4.7/Phaser.ScaleManager.html#.USER_SCALE

this.scale.scaleMode = Phaser.ScaleManager.USER_SCALE;

//resize because it's better than any of the phaser provided resizes

window.addEventListener('resize', (e: Event) => this.mobileResizeCallback(this.game.scale));

this.game.scale.onSizeChange.add(() => {

   this.game.state.getCurrentState().resize();

}, this);

this.mobileResizeCallback(this.game.scale);

mobileResizeCallback ресайзит игру на основе ratio девайса.

Для события изменения размера окна - назначен слушатель, для каждого стейта отдельный.

this.game.scale.onSizeChange

Масштабирование графики

Кроме описаного выше масштабирования, посредством плагина, используются так же различные кастомные методы.

в FunnySplash.ts используется scaleFactor, который вычисляется на основе соотношения сторон девайса.

public getScale(): number {

   if (!this.game.device.desktop) {

       if (this.game.width < this.game.height) {

           return (this.game.width / 6 * 4) / FunnyGamesSplash.IdealWidth;

       } else {

           return (this.game.width / 6 * 3) / FunnyGamesSplash.IdealWidth;

       }

   } else {

       if (this.game.width < FunnyGamesSplash.IdealWidth * 1.5) {

           return (this.game.width / 6 * 4) / FunnyGamesSplash.IdealWidth;

       }

   }


   return 1;

}

Полученое значение используется как для масштабирования, так и для относительного позиционирования.

Пример

this.preloadText.x = this.game.width / 2;

this.preloadText.y = this.game.height / 2 + 230 * this.scaleFactor;

this.preloadText.scale.set(this.scaleFactor);

При построении игровой сцены, для масштабирования объектов используются магические числа (возможно не дописан код)

Примеры:

this.deckCard.scale.set(1.35, 1.35);

back.scale.set(2, 2);
