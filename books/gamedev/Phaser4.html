Phaser проект

Node.js ››

ИНСТРУКЦИЯ: http://divillysausages.com/2015/06/09/using-phaser-with-visual-studio-code/


Установка:


1. Посетить сайт TypeScript http://www.typescriptlang.org/


2. Скачайте и установите Node.js https://nodejs.org/en/


3. Установить TypeScript для Node.js
       npm install -g typescript


4. Скачайте и установите Visual Studio Code https://code.visualstudio.com/

Создание проекта:


1. Создать папку проекта [project]. В ней создать папки:

       [bin]

               [assets]

               [js]

       [src]

       [tsDefinitions]


2. Создать файл tsconfig.json и вставить код

{

       "compilerOptions": {

               "target": "ES5",

               "module": "commonjs",

               "sourceMap": false

       }

}

3. Копируем файлы Phaser из папки ...\phaser-2.4.6\typescript\ в папку ...\tsDefinitions\

p2.d.ts

phaser.comments.d.ts

phaser.d.ts

pixi.comments.d.ts

pixi.d.ts


4. Скопиравать файл phaser.js в папку ...\bin\js\


5. Скопировать файл ds_logo.png в папку ...\bin\assets\


6. Создать файл game.ts в папке ...\src\  и вставить код.


7. Создать файл index.html в папке ...\bin\  и вставить код.


8. Отладим проект Ctrl+Shift+B в результате чего в папке ...\.vscode\  будет создан файл tasks.json в котором необходимо извенить строку "args": ["--out","bin/js/game.js"]


9. Отладим проект Ctrl+Shift+B в результате получим скомпилированный файл game.js

Создаём систему запуска сервера


1. Нажимаем F5 в результате чего будет создан в папке ...\.vscode\  файл launch.json


2. В файле должно быть описаны следующие опции

       "program": "app.js"

       "stopOnEntry": false

Устанавливаем node-static


1. Чтобы выполнить установку необходимо открыть папку проекта и удерживая shift открыть контекстное меню в котором выбрать "Открыть окно команд" и  выполнить команду: npm install node-static


2. Создать в папке проекта  файл app.js в который вставить код работы сервера.


3. Нажимаем F5 открываем браузер и вводим адрес localhost:5858
