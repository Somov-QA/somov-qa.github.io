Начало работы с проектом

Инициализация

               $ git init

Добавляем удалённый репозиторий

               $ git remote add origin https://...

Получаем клон репозитория

               $ git clone https://...


Рабочий процесс на каждый день


Получаем актуальные данные с сервера (пул).

               $ git pull origin

Получаем задание. Создаём ветку.

               $ git branch task12345

Переключаемся в новую ветку

               $ git checkout task12345

Добавляем изменения

               $ git add -A

Комментируем

               $ git commit -m “...”

Отправляем данные на сервер (пуш)

               $ git push origin task12345


Завершающий этап работы над задачей (слияние)


Переключаемся в ветку мастер

               $ git checkout master

Выполняем слияние веток. (новая ветка сливается с мастером)

               $ git merge testing

Удаляем новую ветку

               $ git branch -d task12345

Отправляем данные мастера на сервер (пуш)

               $ git push origin master
