Группа - Phaser.Group

Phaser ›› TypeScript ››

var sprite: Phaser.Sprite;


var group: Phaser.Group = new Phaser.Group(this.game, this.stage);

this.group.addChild(sprite);


При удалении группы нужно удалять её дочерние объекты!

this.group.removeChildren();


Можно получить объект группы по его индексу

var obj: any =this.group.getChildAt(0);

obj.inputEnabled = false;
